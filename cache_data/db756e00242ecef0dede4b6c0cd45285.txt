东方永夜抄 ，担任永夜抄的二面Boss。 她是被称为夜雀的鸟妖怪。 尽管夜雀 原本是只闻其声、不见其形的妖怪，但她却很爱表现自己 [ 大·东方Project展 ] ，并 非常喜欢热闹、吵闹的事情 [ 花映塚 ] 。 在永夜抄之后，还登场于 东方花映塚 中，在官方出版物中也时有作为摊贩出场。 生活状况 她 只会于深夜出没 [ 永夜抄 ] ，在 黑暗的夜道和无人的森林用歌声迷惑人类使人迷路 [ 史纪 ] ，进而袭击被迷惑的人。 在白天她则 往往隐藏起来 [ 史纪 ] 。 她可能是文盲。 [ 文花帖（书籍） ] 不过，在讲述雀酒的制作传说时，她曾经拿着一本书对着看才将传说的后半部分讲完。 [ 三月精O ] 她在 兽道 摆摊开烤八目鳗店，将八目鳗卖给被自己变成雀盲眼的人。最开始她用 挂着红灯笼的移动货摊，会混入鳗鱼、泥鳅和解除施加的夜盲症来骗人 [ 文花帖（书籍） ] 。 博丽神社 开始举行各类祭典后，她也常常会 在活动时去神社开烧烤摊 [ 三月精O 、 茨歌仙 ] 。 她一天到晚用鸟头唱歌，歌声即使在白天也没有停息。 [ 花映塚 ] 她的歌声意外地浮华，拥有一种很不像妖怪之曲的优秀拍调，因此受到年轻人欢迎。 [ 史纪 ] 在“鸟兽伎乐”成立后，她会和 幽谷响子 一起 在深夜定期举办游击演唱会，穿着奇特的服装边甩着头发边唱歌，发出只能让人理解为是噪音的充满杂音的爆音，唱传递性很强的歌词与使用绝大音量的音乐，因此被很多人抗议 [ 口授 ] 。 角色能力 米斯蒂娅拥有 用歌声诱人发狂程度的能力 和 用歌声迷惑人程度的能力 。 她的歌声会令人类的判断神经变得迟钝。 [ 永夜抄 ] 被她的歌声迷惑的时候迷路的话，就会被妖怪袭击。 [ 史纪 ] 听了她的歌，人类会发狂。 [ 花映塚 ] 只要听了她的歌，多难的歌都想唱唱试试。听了这种歌会变得奇怪，三半规管也会失常。 [ 花映塚Music Room ] 她 拥有让人类变得夜盲的能力 [ 史纪 ] 。 她 借此来隐藏自己 [ 永夜抄 ] ，因此她一度几乎 没有被见到过外貌 [ 史纪 ] 。 外貌特征 白天的时候接触的目击报告很少，据说是一个拥有大型羽翼的人型妖怪 （×1如果一开始就看到这样的外表的话，就不会被叫做夜雀了吧。） 。 米斯蒂娅的歌是意外地浮华，拥有一种很不像妖怪之曲的优秀拍调。因为这点，在年轻人之中受到欢迎 （×2尽管不应该受欢迎的。） 。 相对的，歌唱妖怪的老一辈都是说：“最近的年轻妖怪就知道吵闹，成天唱些不明所以的歌，很让人困扰。那不就是噪音吗。”这般的话。 同时，为了使得在被歌声引诱时看不清周围，她拥有让人类变得夜盲的能力。仅靠着有限的灯光和歌声在夜道上行走的话，这样下去会被这个妖怪袭击的。 目击者报告 我还以为普莉兹姆利巴乐团雇佣了歌手，但我靠近了之后也没发生什么好事。（匿名） 和普莉兹姆利巴乐团不同，她对于人类的友好程度是很糟糕的。 就算听到歌声了，也不要因为感兴趣而走近。 对策就是，不要独自一人在黑暗的夜道上行走。不仅限于这个妖怪，也是适用于其他妖怪的最基本的对应方法。 那么，夜盲的对策应该怎么做？其实，吃八目鳗是个好办法。每天吃的话，就会逐渐变好的吧。 要降伏这个妖怪的话要怎么做才好？因为在夜晚几乎没有得胜的希望，所以降伏要在白天进行，但由于在白天往往隐藏起来所以那也不是件容易的事。最有效的降伏方法是，傍晚的时候设好陷阱 （×3粘鸟胶会很有效。）
