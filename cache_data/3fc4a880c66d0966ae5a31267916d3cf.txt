操纵冰程度的能力 （辉针城、天空璋、人妖名鉴常世篇、幻存神签） 琪露诺（官方角色，新作） - 新作角色。雾之湖畔的冰精，自认最强的笨蛋 角色介绍 琪露诺是东方Project系列中的官方角色，首次登场于 东方红魔乡 ，担任红魔乡的二面Boss。 她是冰之 妖精 。 她 脑子有些笨笨的 [ 红魔乡设定文档 ] ，和其他妖精一样天真，喜欢恶作剧。 在 东方花映塚 的 介绍图 中，琪露诺的位置⑨被标注为“バカ”（笨蛋），这之后⑨、笨蛋、baka就被用来称呼琪露诺。 铃仙·优昙华院·因幡 评价她波长很短，对应着她容易冲动。 [ 花映塚 ] 她从不考虑自己的力量就去恶作剧，因而每次都会吃苦头。对于全部妖精来说，无论吃多少苦头也不会反省一点,下次就继续恶作剧了。 [ 花映塚角色设定 ] 在红魔乡后，琪露诺也多次在官方游戏和出版物中出场，并在 东方花映塚 、 东方非想天则 、 妖精大战争 和 东方天空璋 中作为自机出场。 生活状况 平时以 雾之湖 作为根据地。 有时会在 雾之湖 一带 毫无顾忌与防范地睡觉 [ 史纪 ] 。 稗田阿求 猜测她的家说不定在水中。 [ 史纪 ] 但实际上，她的家是雾之湖旁边的一个冰屋。 她是湖附近妖精们中的领袖般的存在，力量也比其他妖精强。她是属于好战型的，所以必须留神。 [ 史纪 ] 但是她其实 和杂鱼没什么大区别 [ 辉针城设定文档 ] 。 她自认为是 最强的 [ 花映塚 ] 。 喜欢玩一种把青蛙冻住后放到水里解冻的恶作剧游戏。称之为 “结冰的修行” [ 文花帖（书籍） ] 。但是 每玩三次大概就会有一次失败，青蛙就会悲惨地支离破碎 [ 红魔乡设定文档 ] 。 博丽神社 开始举行各类祭典后，她常常会 在活动时去神社开刨冰摊 [ 东方三月精 、 东方茨歌仙 ] 。 角色能力 琪露诺拥有 操纵冷气程度的能力 和 操纵冰程度的能力 。 琪露诺的体温很低，一直在释放冷气的她周围总是很寒冷。她接触的东西能瞬间变成冰，比普通的妖精更危险，但对火把的火之类热而不可触摸的东西很弱。 [ 史纪 ] 铃仙 认为琪露诺制冷是通过抽走周围的热量，所以体内会很热。 [ 非想天则 ] 琪露诺声称自己冻青蛙 是结冰的修行 [ 文花帖（书籍） ] 。 作为妖精是 不会死 [ 史纪 ] 的。 但 四季映姬·夜摩仙那度 称妖精干了太多超出自己领域的事情、惹的麻烦有点太多的话，也有可能受到依靠自然之力所无法治愈的伤害进而死亡。 [ 花映塚 ] 外貌特征 琪露诺属于 身长低 [ ZUN的邮件答复 ] 的角色。 她有着一头浅蓝色短发，脑后扎着大蝴蝶结，在花映塚、非想天则中为绿色，在其他作品中为蓝色。 她有着蓝色的眼瞳。 她身穿蓝色连衣裙，裙子下摆为白色，与蓝色交界处为锯齿状，在史纪中裙摆侧面有缺口，并有菱形绳结装饰。裙底穿着白色灯笼裤。领口系着红色领结，在天空璋和人妖名鉴 常世篇中，领口还戴着一朵向日葵，身上缠着牵牛花。 她背后生有三对冰晶状的菱形翅膀（在红魔乡、非想天则中并没有明显分开，在辉针城中仅能看见两对）。 在妖妖梦、求闻史纪中脚穿白色短袜，在人妖名鉴宵暗篇中脚穿蓝色玛丽珍鞋，在其他作品中为裸足。 红魔乡
