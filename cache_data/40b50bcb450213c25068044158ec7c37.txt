洩矢诹访子（官方角色，新作） - 新作角色。土著神的顶点，统领祟神的腹黑蛙神 角色介绍 洩矢诹访子是东方Project系列中的官方角色，首次登场于 东方风神录 ，担任风神录的EX面Boss。 她原本是生活在 外面世界 的土著神，是 八百万神 之一；现在则是 守矢神社 供奉的神之一、 山神和住在山里的各种各样的神明的领导者 [ 风神录设定文档 ] ，并统领着以 御社宫司神 （又名 赤口神 ）为代表的祟神。 御社宫司神是 负责诞生、农业、军事，各种各样的事情的作祟神，也是只要受到一点轻视马上就会降下神罚将对方杀掉的恐怖神明 [ 风神录设定文档 ] ，同时也是 丰穰之神 [ 口授 ] 。 她的 性格初看好像很温厚，总是装出一副呆呆的样子。但是无法看透在想什么，真实性格或许很坏 [ 口授 ] 。她还有些任性妄为， 多次不告诉神奈子与早苗擅自进行自己的计划 。 在风神录之后，她也作为敌机和可选自机出现在 东方非想天则 中，并多次为 东风谷早苗 提供支援和在早苗的结局中出场，在官方出版物中也时有出场。 生活状况 诹访子通过管束只有她能控制住的御社宫司神，曾作为神的同时作为王建立起洩矢之国，但在抵抗失败后将王国交给了 八坂神奈子 ，不过之后一直都在实质上继续支配着王国。 [ 风神录设定文档 ] 如今诹访子居住在 守矢神社 ， 不怎么离开神社 [ 口授 ] 。 作为山神，她统率着山中的神明。 [ 风神录设定文档 ] 她平时以人形态示人，祭祀请神之时，则常以青蛙的形态出现。 [ 口授 ] 角色能力 诹访子拥有 创造坤程度的能力 。 使用这项能力，诹访子能 改变地形，还能随心所欲地使土地变成盛产谷物的良田，或是变成寸草不生的诅咒之地 [ 口授 ] 。 诹访子会 用这种能力开凿洞穴等 [ 口授 ] 。 诹访子还 自称妖怪之山如果喷发，她“会调整喷发的日子的” ，也就是说应该有着能够调整喷发程度的力量。 [ 口授 ] “坤”是八卦之一，表示地，与神奈子的“控制乾程度的能力”相对应。 诹访子有统领祟神，尤其是管束与控制御社宫司神的力量，这种力量是她 取得 信仰的重要来源。 事实上，诹访子整平大地也是靠驱使祟神完成的，并且她掌管的祟神同样能使土地长出谷物或变成不毛之地。 [ 口授 ] 诹访子在 东方智灵奇传
