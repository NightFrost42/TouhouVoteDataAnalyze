，但是其性质非鬼非神亦非精灵， 是天邪鬼的一种 [ 绀珠传Music Room ] ，也 是位居天邪鬼之上的神灵 [ 绀珠传设定文档 ] 。 她也同时拥有天津神、国津神双方的属性。 [ 绀珠传访谈 ] 由于能力的缘故，她性格寡言， 甚至在梦中也是如此 [ 噩梦日记 ] 。 她在官方游戏与出版物中登场次数不多。 生活状况 探女在 月之都 担任高官，却不会经常在外界面前出现。 在绀珠传前后的一段时期，她多次试图利用幻想乡，使幻想乡每每 陷入 [ 东方深秘录 、 东方绀珠传 ] 或 差点陷入 [ 东方文果真报 ] 危机。 角色能力 探女拥有 从口中说出便可使事态逆转程度的能力 。 一旦她说了某个事态，尤其是说有关的人，这个事态就会向反方向发展，本能成功的事会失败，坏事则会变成好事（并不是对自己而言的）。 利用她的能力制作的力量石，有着可以用话语改变世界的力量，只需要在力量石附近，就可以把不应存在的流言（都市传说）具现化。 她的能力可以让 《文文春新报》 中夸张的部分在发售后成为真正的现实。 这个能力就是 她身上存在的天邪鬼「欺诈」的部分 [ 绀珠传访谈 ] 。因此， 她说话有时会使事情向好的方向发展，有时也会使事情向坏的方向发展，但不论如何，世间一定会发生变化，而这种变化可能对自己有益，也可能对对方有害。 [ 绀珠传访谈 ] 外貌特征 探女有着一头银灰色及肩短发，略卷，扎成公主辫，在脑后收束为一条小辫。 她有着红色的眼瞳。 天邪鬼只会说自己反对的事情，但她会通过自己说的事情让世界向反方向发展。 如果她想做成什么事情，那一定悉数失败， 如果有什么坏事发生，那她的能力一定可以使事态好转。 但那并不意味着一定会对自己好转。 正所谓，祸从口出的神灵。 月之民的贤者们，为了防止纯狐的攻击导致的污秽侵蚀，将月之都冻结了。 与此同时，他们令月之民前往梦境避难。 梦境中的都城，正如大家所想，是往日的月之都。
