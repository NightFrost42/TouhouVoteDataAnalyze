，果没有文在新闻取材上的蛮干劲，也没有文那么傲慢。 [ DS ] 在DS之后，她在官方游戏与出版物中登场次数不多。 生活状况 在 文花帖DS 的故事之前，从未外出取材，而是用念写的能力拍摄照片，再根据照片撰写新闻。 [ DS设定文档 ] 由于获取情报的速度慢与新鲜度低，所以《花果子念报》人气很低。 果对新闻的观念是， 反复推敲的文章才能赋予采访以生命力 [ DS ] ， 与照片搭配得当的文章能进一步体现照片的魅力 [ DS ] 。 东风谷早苗 认为果的 文章攻击性有点强 [ GoU ] 。 在文花帖DS之后的官方漫画中，可以看到果外出取材的身影。 [ 茨歌仙 ] 她所用的相机是 河童制造的 [ DS ] ， 可以连拍，而且换胶卷的时间很短 [ DS ] 。 角色能力 果拥有 使用念写程度的能力 。 念写意为“将心中所想全部浮现于画像上”， 是接近于千里眼的能力，但是可以影像化 [ 口授 ] 。 果念写的能力体现为 在果所持有的照相机中输入关键字，便会有与其相关的照片显现 [ DS设定文档 ] 。 念写出的照片可以显示在电脑上，还可以用打印机打印出来。 [ 智灵奇传 ] 此外，果的相机有 连拍功能 [ DS ] 。 外貌特征 果有着一头棕色齐耳短发，用紫色蝴蝶结扎成及胸的双马尾。头戴小型紫色六角帽（头襟）。 她有着棕色的眼瞳。 她上身穿着短袖衬衫，在DS中为淡粉色，在人妖名鉴中为白色。衬衫的袖口系着蝴蝶结，打着领带，蝴蝶结和领带在DS中为黑色，在人妖名鉴中为深蓝色。领子打着粉色褶子。 她下身穿着紫黑相间的格子短裙。在DS中，右腰挂着一个棕色荷包。 她穿着黑色及膝袜和天狗标志性的一齿高木屐，腿上绑着带子。木屐和带子在人妖名鉴中为红色，而在DS中，木屐为棕色，带子为紫色。 左手持翻盖手机样子的念写相机，相机镜头下有一个红色爱心图案。相机的翻盖转轴挂着一只毛笔，DS中挂在右侧，人妖名鉴中挂在左侧。 文花帖DS 人妖名鉴 人际关系 果和 射命丸文 同为 鸦天狗 ，果将文视为新闻方面的竞争对手，文也会 故意歪曲果的报道来抨击果的《花果子念报》 [ 文果真报 ] 。 作为鸦天狗，果是 饭纲丸龙 的下级。 [ 史纪 ] 登场剧情 在 东方文花帖DS 中，着眼于《文文。新闻》，为了知道《文文。新闻》具有魅力且不会和其他新闻撞车的原因而跟踪 射命丸文 ，发现这原来是因为文不是将事件写成新闻而是在引发事件。 于是，果向文下了宣战布告，决定不使用念写而去收集实际的题材，打算用文的拍摄对象写出更优秀的文章，从而在新闻大会上取得优胜。 随后，文和果进行了几次取材大战，果在文面前完全不是对手。不过，果坚持认为文章的内容胜于事件的品质，并认为自己有可能超越文。 之后，文取材大战中让着果，乃至最后放弃进行拍摄，于是果对取材信心大增，便第一次出门四处取材。 在 弹幕天邪鬼 中，参与了对 鬼人正邪 的围剿。 在 东方天空璋 中，文需要果的念写能力以反击 摩多罗隐岐奈 ，而找果商量在调查四季异变时意识到了天狗的危机。在商量时果发现了文背后的门。 在 东方智灵奇传 中，用能力拍到 怨灵 所引发的一系列事件的照片，并在秘密据点将照片打印了出来，用这些照片作为素材写了一期攻击文因过于小瞧怨灵终遭报应的花果子念报。 之后， 古明地觉 来到果的秘密据点。果告诉觉自己已被怨灵察觉到，因此怨灵才能用果的报纸来骗文。 觉走后，果用能力拍到并打印出怨灵在 守矢神社 的照片，便推测怨灵下一个目标是 命莲寺 ，于是宣称怨灵在命莲寺。 文和 博丽灵梦 来找果询问证据，果展示了照片并说明了自己的推理。 不过，当二人去命莲寺进行调查后，果在秘密据点用能力拍怨灵的照片，却只能显示“图像已损坏”，也只能打印出一片漆黑。 官作出场记录 角色出场标准 黑色表示该角色在此处作为主要角色出场 灰色 表示该角色在此处仅作为背景出场 暗灰 表示该角色在此处并未实际出场 登场信息 - 一设资料 东方文花帖DS 东方文花帖DS立绘 设定文档 ○当代的念写记者 姬海棠果 Himekaidou Hatate 种族：鸦天狗 能力：使用念写程度的能力 她虽然是新闻作者，但从未外出取材过。 因为她是用念写拍摄照片，并借由照片来撰写报道。 所谓念写，是指在果所持有的照相机中输入关键字，便会有与其相关的照片显现之事。 果觉得这很方便，于是认为没有必要从家中走出去了。 她所发布的报纸，《花果子念报》没有什么人气。 想来也是当然的。因为她所写的东西都是些无论哪里都已经传开了的事情。 念写在获取情报的速度与新鲜度上存在问题。 就在这时，她着眼于文所发布的《文文。新闻》。 “明明是这样胡编乱造的新闻，为何还能具有魅力呢？ 为何不会与其他新闻撞车呢。” 她为了探取这个秘密而跟在了文的后面。 于是她知道了文的秘密。 并不是将事件写成新闻，而是文自身在引发事件。
