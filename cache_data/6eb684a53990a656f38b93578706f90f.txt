ZUN认为天子在形象上与《西游记》里的孙悟空比较接近。 [ 人妖名鉴 ] 天子帽子上有桃子装饰。 天子袖口的花纹形似金箍。 天子因为偷吃仙丹而被流放。 [ 凭依华 、 茨歌仙46话 ] 天子在非想天则中被 八云紫 打败后， 紫讲了一遍大闹天宫的故事 [ 非想天则 ] 。 她有着 温室娇女般的性格，任意妄为、自鸣得意 [ 绯想天设定文档 ] 。因此， 只有捧着顺其心意才能让她好好听话 [ 茨歌仙45话 ] 。 同时，她 天性贪玩 [ 绯想天设定文档 ] 、 害怕寂寞 [ 交叉评论 ] ，因而羡慕幻想乡里人类与妖怪的生活。 在绯想天之后，她还作为自机登场于 东方凭依华 中，并在官方出版物中时有出场。 生活状况 在 完全凭依异变 前不久，天子因为偷吃 天界 宴会上的丹药被放逐到地面上。 [ 东方凭依华 、 茨歌仙46话 ] 角色能力 天子拥有 操纵大地程度的能力 。 作为“比那名居一族”，她能够操纵要石，通过插下和拔出要石平息或引发地震。 她能够以此引发地表下沉、土崩石摧级别的灾难，甚至动摇整个幻想乡的地基。 [ 绯想天设定文档 ] 绯想之剑是 天界 的道具，只有天人能操纵，被天子带走作为自己的武器。 绯想之剑能够将对手的气质雾化并转换为某种具体形态（天气），能洞察对手气质的弱点并转化为性质缠绕，准确无误地将弱点击中。 [ 绯想天设定文档 ] 天子把这种能力称为“绯想之术”， 认为 秦心 夺取感情的能力与自己的绯想之术很像 [ 凭依华 ] 。 此外，天子虽不具备天人应有的品行，但具备天人的许多特质。 肉身十分强韧， 刀子都刺不进天人的身体 [ 绯想天 ] ， 天人的肉对妖怪而言是剧毒 [ 史纪 ] 。 拥有极强的气运， 天子即便和贫穷神 依神紫苑 相处幸运也不会有所削减 [ 凭依华 ] 。 以仙术为基础的封印对天人没有意义，天人能够直接发现仙界的入口。 [ 茨歌仙46话 ] 对于 气息 有敏锐的感知力，能够感知到 “死亡的味道” [ 茨歌仙46话 ] 、 “地狱的血腥味” [ 茨歌仙48话 ] 等。 外貌特征 天子有着一头蓝色及腰长发，末端稍卷。头戴一顶黑色礼帽，帽子上装饰有 可以吃的桃子 [ 东方凭依华 ] 和桃树叶。 她有着红色的眼瞳。 名居一族是地上的神官，死后被称为名居守，在山中池 水边上的祠堂里被视为神明祭拜。 名居守至今也静静地守护着幻想乡。 而与此同时，在生前作为名居守部下的比那名居一族，也 因这个功绩而得到褒奖，得以在天界居住。 但是，比那名居一族与其他的天人不同，并没有积累过足够的修行，仅 仅是供名居守差遣，所以不具备天人应有的品格，在天界被称为不良天 人。 比那名居地子只是沾了父母被名居守差遣的光而附带着成为 了天人的小孩子罢了。 地子在成为天人的时候，把名字改为了天子。 天子并不喜欢自己在天界的处境，多次偷偷观察着幻想乡，对村中的人 类和乡里的妖怪们怀有羡慕之意。 然后有一天，幻想乡的妖怪们以自己的力量引发了异变， 在幻想乡境内引发骚动并以此为乐。天子看在眼里，不禁沉思。 「我，对无聊的天人生活已经忍无可忍了！ 就凭我的话，也是可以引发个异变的」 于是天子离开了自己的家。
