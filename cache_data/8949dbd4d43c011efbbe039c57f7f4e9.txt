蓬莱山辉夜 日文名 蓬莱山　輝夜（ほうらいさん　かぐや） 英文名 Houraisan Kaguya 种族 月人 能力 操纵永远和须臾程度的能力 （永夜抄、求闻史纪、儚月抄、人妖名鉴、幻存神签） 蓬莱山辉夜（官方角色，新作） - 新作角色。永远和须臾的罪人，家里蹲的辉夜姬 角色介绍 蓬莱山辉夜是东方Project系列官方作品中的官方角色，首次登场于 东方永夜抄 ，担任永夜抄六面Final B的Boss。 辉夜就是「竹取物语」里面登场的那位“辉夜姬”。 [ 人妖名鉴 ] 她是月之公主， 永远亭 的主人，拥有 操纵永远和须臾的能力 。 来到地面后，通过学习她对地上的知识已经有一些了解，但是也有展示出不足的一面。 [ 儚月抄 ] 她性格就是像二次创作里常见的那种宅家少女的形象，有着对外界没有兴趣的大小姐气质。 [ 幻想巡礼 ] 在永夜抄之后，辉夜在之后的官方游戏与出版物中时有出场。 生活状况 辉夜在被流放到地上前曾居住在 月之都 ，在杀死月之使者后的一千多年里隐居在 迷途竹林 内的 永远亭 中， 如今不再隐居，过着普通的生活 [ 永夜抄设定文档 ] 。 她 每年都会举行一次月之都博览会“月都万象展” [ 史纪 ] ，展示各种关于月之都的物品和资料甚至月兔捣年糕表演，大受好评。 为了打破无事可做的无聊生活，她 把养护盆栽当成了自己的工作 [ 儚月抄 ] 。 她会在 藤原妹红 带路下 定期溜出永远亭 [ 智灵奇传 ] 到 人类村落 游玩。 角色能力 辉夜拥有 操纵永远和须臾程度的能力 。 这是一种 对普通人类来说过于强大的、操纵时间的能力 [ 史纪 ] 。 她能集合人类无法感知的一瞬间，以此来使用和行动。集合了须臾的时间也是在时间中流动，人类们却完全无法感知，利用这时间就能弄出复数不同的历史来。 [ 史纪 ] 她能施下永远的魔法， 停止一切历史的行进，拒绝任何污秽以及变化 [ 儚月抄 ] ， 就算有活动，也和停止时间一样 [ 史纪 ] 。于是，这种魔法能使 生物停止成长、食物放多长时间都不会腐坏、任何器皿无法被打破、泼出去的水收回盆中 [ 儚月抄 ] 。 使用能力，她 能够创造出地上世界不可能存在的，毫无污秽的永远 [ 儚月抄 ] 。 由于拥有着永远之力，她即使死了亦能马上转生。 [ 永夜抄设定文档 ] 辉夜也可以制造出蓬莱之药。 [ 永夜抄 、 永夜抄符卡说明 ] 辉夜制作的蓬莱之药被称作故事的一个要点 [ 永夜抄符卡说明 ] ，也就是被永琳送给老翁的蓬莱之药。 不过，辉夜制造的蓬莱之药是 辉夜用自己名字命名的 [ 永夜抄符卡说明 ] ， 而过往被称为蓬莱之药的不老不死药的名字并不是辉夜取的 [ 儚月抄 ] ，所以辉夜制造的蓬莱之药和永琳或月兔所制造的蓬莱之药（永琳送给老翁的药当然也是永琳或月兔在月之都制造的）似乎又不是同一种药。 她由于 服下了蓬莱之药 [ 儚月抄 ] 而不老不死，同时会 产生和人类同样的污秽 [ 儚月抄 ] 。具体效果请参照 相关考据 。 外貌特征 辉夜有着一头黑色及膝长发，留着姬发式，刘海过眉。 她有着黑色的眼瞳。 她上身穿着粉色和风上衣，胸前用白色丝带扎着领结，衣服扣子是一排白色小领结。两袖长到能藏住手，上有橙黄色图案，左袖为遮住月亮的云，右袖为花札“芒与月”。 她下身穿着深红色和风长裙，裙摆拖到地上，并有白色荷叶边。裙子上有金色的樱花、竹子、枫叶、梅花的描边图案。 她脚穿白色短袜和皮鞋，袜口打着褶子。 在永夜抄中，身边从脚下起按顺时针顺序依次围着燕的子安贝、蓬莱玉枝、佛的御石钵、火鼠的皮衣和龙颈之玉。
