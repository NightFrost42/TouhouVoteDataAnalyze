古明地觉（官方角色，新作） - 新作角色。因会读心而被惧怯的觉，管理怨灵的地灵殿主人 角色介绍 古明地觉是东方Project系列中的官方角色，首次登场于 东方地灵殿 ，担任地灵殿的四面Boss。 她会 读心 的觉妖怪，被 委任管理 旧地狱 - 灼热地狱遗址 的怨灵 [ 口授 ] ，是遗迹上方的别墅 地灵殿 的主人。 然而觉本人却是一个 和蔼、老实、有些内向的妖怪 [ 地灵殿 ] ， 稍有幻想情调并喜爱动物 [ 地灵殿Music Room ] 。 她有很多宠物，读心的能力让她可以和动物们交流，因此 她也很受动物们的欢迎 [ 口授 ] ，但她自己却觉得 宠物在避着她 [ 地灵殿 ] 。 在地灵殿之后，觉在官方游戏和官方出版物中都时有出场。 生活状况 觉居住在 旧地狱 的 地灵殿 中，作为被畏惧与厌恶的存在过着自我隔离的生活。 [ 口授 ] 基本不会有人拜访她，她也几乎不主动出门。 因为她的妹妹终日在外游荡， 她偶尔会到地面上寻找妹妹 [ 凭依华 ] 。 平日在家看书写书，尤其是喜欢带有丰富心理描写的故事。 [ 口授 ] 至于各种日常杂务乃至管理 灼热地狱遗址 的工作，那些都被她交给了宠物们。 [ 地灵殿设定文档 ] 她还负责监视地底的家伙，防止他们出现在地上。 [ GoU ] 另外，她也会作为侦探出面调查旧地狱的各种事件。 [ 智灵奇传 ] 她是为了找出并逮捕 宫出口瑞灵 而成为侦探的。 [ 智灵奇传 ] 角色能力 觉拥有 读心程度的能力 ，她可以知晓他人当下的脑中所想，甚至是探索他们的记忆。 作用的对象可以是任何人、妖怪、动物甚至灵魂。 但这一能力也有局限性，例如： 会因为距离太远而读不到。 [ 地灵殿 ] 无法知晓他人已经遗忘的内容。 [ 地灵殿 ] 读了太多人类的心之后会犯恶心。 [ 智灵奇传 ] 不能像 古明地恋 一样，可以渗透到深层的潜意识。 她因为这项能力而成为 旧地狱 里最被惧怕，最被讨厌的妖怪，连怨灵都对她避之不及。 然而，她对自己的能力十分满意，认为它十分 美妙，是不管对手是谁都能够适用的最优秀的力量 [ 交叉评论 ] 。 和她在一起的宠物会逐渐变强，成为强大的妖怪。 [ 地灵殿 ] 她 并不怎么擅长战斗 [ 地灵殿Music Room ] ，但是比一般的觉更 好战 [ DS ] 和 敏捷 [ DS ] 。 外貌特征 觉有着一头齐脖短发，在地灵殿中粉紫相间，在人妖名鉴中为粉色。头戴黑色发卡，上有黄色心形的装饰，且连接着第三只眼的一部分。 她有着紫色的眼瞳，在地灵殿和求闻口授中闭着右眼。 她上身穿着淡蓝色的上衣，同样有心形的装饰。上衣的衣领和袖口处为粉色，有褶边装饰。扣子也为黄色心形。 为了体现“读心”与“可爱”，ZUN在设计她时有意加入了大量心形饰品。 [ 地灵殿采访 ] 在到处都是被嫌恶者的旧地狱中，在这群体中有一类特别惹人讨厌的妖怪，那就是觉 2 了。尽管当时是由于无法跟人类以及妖怪和谐共处从而逃到旧地狱去的，但结果，还是被旧地狱的妖怪所讨厌。 那理由，就是来自她的读心能力。无论谁都不想要跟会读心的家伙碰面啦。被读心这种事不仅是等同自己赤着身子跟她说话，而是意味着连普通地对话也无法做到。她在旧地狱中一个名为地灵殿的地方把自己完全隔离，放弃跟任何人进行交流。 地灵殿基本上是不会有客人的。特意前来探访她的人更不会有。 然而，她却很受无法说话的动物们所欢迎。因为读心的能力，对于动物来说可以说是独一无二的能力。就因为这样，地灵殿栖息着很多动物 （×1 博丽灵梦小姐曰「她们只是非常好用的奴隶」。） 。 她对无法说话的灵魂一样可以读心。正因为这样连怨灵都对她避之不及。怨灵和动物不同，自己的所思所想还是想要凭借对话传达出来的。这理由不用多说，因为「会说话的生物，全部都是有表里两面」的缘故吧。 为了对怨灵害怕她这件事有效利用，她如今负责管理着仍残留在灼热地狱遗址内的怨灵。由于无法对她隐瞒任何事，也是等于不会有人敢去逆她的意去作乱。顺带一提，地灵殿就是建在灼热地狱遗址的正上方。 她有一个妹妹。 她的宠物阿燐 她有一只名唤火焰猫燐的宠物。似乎都是一直叫阿燐。 由于阿燐能做到跟怨灵交谈，所以一直作为她的右手帮助她的样子。根据前述，怨灵饥渴于以对话与人交流的缘故，所以她们反而会去投奔阿燐而不是觉。因此也有实际上管理怨灵的人是阿燐的说法。 虽说是管理，但也基本就是放着不管，偶尔平衡下数量的程度。 她的宠物阿空 灵乌路空也是她的一只宠物。她都是叫阿空的。 作为在地狱诞生的地狱鸦，她是一个能平静置身于灼热地狱遗址之热中的存在。因此她一直负责对灼热地狱遗址进行温度调节。 虽然阿空是个鸟脑袋，但应该擅长于不需要多动脑子的单纯劳动的作业。 她跟阿燐非常地要好。觉的宠物们都很仰慕着觉，因此伙伴意识很强。 地灵殿 管理着灼热地狱迹遗址她所身处的别墅。地板还装有暖气。只有古明地姐妹住在这个宽敞得要命的地方 （×2 地狱比起地上要广敞得多。旧地狱当然也不例外。） 。完全没有访客。 觉即使在地灵殿内也不会怎样动身，基本上都在自己的房间看书、写书的样子。特别喜欢带有丰富心理描写的故事。读书、写书对于能够读心的人据说是很刺激的体验。在作者不明的故事中，或许就有她写的书。 同时，她的领地内几乎到处都是宠物。就像是动物园的样子。 对策 虽然是令人害怕的妖怪，但幸运的是基本上不会有机会遇到她。 万一真的对峙起来，比起想一些拙劣的计策，还不如立即改变态度比较安全。
