骊驹早鬼是东方Project系列中的官方角色，首次登场于 东方鬼形兽 ，担任鬼形兽的EX面Boss。 她是骊驹的动物灵，是拥有强健的脚力、獠牙和肉体的动物灵组织“劲牙组”的组长。 她的设计参考了传说中曾载圣德太子飞越富士山的“甲斐黑驹”和希腊神话中的天马“珀加索斯”（Pegasus）。她的立绘剪影出现在了鬼形兽的封面上。 她崇尚实力，热衷于正面对决，对于自己的实力非常自信，甚至可以说是盲目自大，对于吉吊领导的“鬼杰组”背后袭人的战斗方式则非常鄙夷。 她认为劲牙组是 畜生界第一的最强最快的组织 [ 鬼形兽 ] 。 她非常尊敬有实力的人，与自机们交手后都表达出 想将自机招纳入劲牙组的欲望 [ 鬼形兽 ] 。 但是她头脑并不是很灵光， 被其他动物灵评价为“肌肉笨蛋”，劲牙组也被评价为“肌肉笨蛋组织” [ 东方鬼形兽 ] 。 ZUN本人也在乐评中认为她是肌肉笨蛋。 [ 鬼形兽Music Room ] 她有着强烈的野心和扩张欲望，不仅想要支配畜生界，还觊觎着人界。 她做出决定就会立即行动，尊重部下的想法和功绩，作为上司还算比较受欢迎。 [ 鬼形兽设定文档 ] 她喜欢 自称为漆黑的 天马 ，遇谁都要强调一次 [ 鬼形兽 ] 。 在鬼形兽之后，早鬼还登场于 东方兽王园 中。 生活状况 早鬼 平时居住在 畜生界 ，作为动物灵组织“劲牙组”的组长 [ 鬼形兽设定文档 ] 率领劲牙组与其他动物灵组织互相争斗，企图支配畜生界。 与其他动物灵组织一样，她以及她领导的劲牙组把人类灵当作奴隶进行奴役。 同时， 依附于组织生存的动物灵也都是组织的奴隶 [ 鬼形兽 ] ，并且早鬼在工作方面对部下很 黑心 [ 鬼形兽设定文档 ] 。 她会为了遇到强者而频繁到地面上。 [ 黑市 ] 她会精心保养她引以为傲的黑发和黑色羽翼。 [ 人妖名鉴 ] 角色能力 早鬼有着 拥有无双脚力程度的能力 。 虽然能力被描述为“脚力”，但她实际上长着一对翅膀，也可以在空中飞行。 早鬼拥有强健的肌肉和力量，一旦开始攻击就不会停手。这也是劲牙组共通的战斗风格。 她的实力受到其他动物灵的尊敬。 庭渡久侘歌 称她为 “最强的动物灵”。 [ 鬼形兽 ] 另外，动物灵能够凭依、占据其他人的身体。 这一点表现在野狼灵、水獭灵、大鹫灵都能占据鬼形兽自机的身体。 早鬼作为动物灵，也有可能拥有这样的能力，但暂时不能确定。 外貌特征 早鬼 实际身高比她给人的印象要矮 [ 人妖名鉴 ] 。 她有着一头黑色及膝长发，在身体侧后方绑成长马尾，头上戴着深棕色牛仔帽。 她有着红色的眼瞳。 她身穿形似骑马服和牛仔服的短袖连衣裙，颜色自上到下分别为蓝、粉、棕，分界处有上蓝下黄的锯齿状和三角形图案，蓝色部分有白色网状细纹，棕色部分有深棕色格子图案，中间开衩。脖子上系着打结的围巾，在鬼形兽、人妖名鉴中为绿色，在兽王园中为白色。 她背后长着一对巨大的黑色翅膀， 翅膀上没有羽毛 [ 兽王园 ] 。 她下身内穿黄色短裙。 她身后有一条黑色马尾巴。 她脚穿棕色长筒靴，在人妖名鉴中脚后跟有马刺。 在 地狱 ， 庭渡久侘歌 等人在水滨迎击住早鬼。随后，自机也来到地狱阻止早鬼。激战之后，早鬼作为EX面Boss被自机击败，对地上的侵攻也随之停止。 在 东方兽王园 中，在 日白残无 的介绍下将 三头慧之子 以怀柔政策笼络进劲牙组，此外还雇佣了 菅牧典 打探鬼杰组和刚欲同盟的动向。 在地狱遇到被传送到地狱的 清兰 发展了劲牙组的野兽新成员。 在决战前，找慧之子做准备运动，随后让慧之子从地上的妖怪手里保护大地。 之后，在 旧灼热地狱 打败 天火人血枪 ，进而前往 旧血池地狱 打败 饕餮尤魔 。之后尤魔说敌人不是自己、不是 吉吊八千慧 、甚至不是 博丽灵梦 ，但是早鬼并没有理解尤魔的意思。 随后，得知八千慧揍了一顿慧之子，便找到并打败 孙美天 ，遗憾于美天不在劲牙组。 不久，得知有人类接近了受伤的慧之子后消失，派出更多野狼灵去监视，随后找八千慧对战来为慧之子报仇，随后与八千慧讨论局势。 之后，尤魔到地狱找早鬼对战，打败早鬼后告诉早鬼自己要去见残无，早鬼才反应过来真正的敌人是残无，便通过慧之子的传送陷阱找到残无，但却以为残无要掌控劲牙组来篡位夺权。残无否认了早鬼的猜测，并声称一切都在自己手掌心里，随后与早鬼对战并输给了早鬼。
