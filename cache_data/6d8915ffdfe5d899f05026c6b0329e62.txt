，担任封魔录的四面Boss；新作初登场作品为 东方红魔乡 ，担任红魔乡的自机。 与灵梦作为游戏的初始界面类似，魔理沙作为整数作游戏图标已经成为东方系列的惯例。 她的种族是人类，职业是魔法使。 虽然 看上去很别扭 [ 风神录设定文档 ] ，但她 容易亲近、有人情味 [ 幻想揭示板 ] ， 和她一起会很有意思，让人心情愉快 [ 史纪 ] ，并且性格 争强好胜 [ 永夜抄Manual ] 不服输 [ 萃梦想设定文档 ] 。虽然她稍有些 扭曲 [ 地灵殿设定文档 ] 、 乖僻 [ 红魔乡设定文档 ] 、 爱戏弄人 [ 史纪 ] 和 心术不良，本性却很正直 [ 萃梦想设定文档 ] 。她虽然 头脑很灵活 [ 红魔乡Manual ] ， 态度看起来也很轻浮，但其实是个爱操心、认真的努力家 [ 大东方展 ] 类型的人。 从 东方梦时空 开始，魔理沙一直作为大部分官方游戏可使用的自机角色登场，在所有官方出版物中也都是主角或常驻角色。她贯穿了东方系列的几乎整个故事线，是东方剧情中 最重要的角色 之一。 生活状况 魔理沙经营着位于 魔法森林 内的“ 雾雨魔法店 ”，她的家雾雨邸就是店铺。 她同时也居住在雾雨邸，在有工作或异变发生时则会出现在 幻想乡 内的各个地方乃至幻想乡外的世界中。 由于魔法店的位置，找到这家店十分困难。 [ 文果真报 ] 她平日里埋头于魔法研究，也经常出入 博丽神社 。尽管 经常出门 [ 永夜抄设定文档 ] ，却 自认为是个家里蹲 [ 兽王园 ] 。 她比较怕冷。 [ 大东方展 ] 她 有收集癖，且在年年加重 [ 妖妖梦设定文档 ] 。 她经常 小偷小摸 。 她会使用 魔法扫帚 、 迷你八卦炉 、自制魔导书等等道具。 魔法扫帚是她自作主张地认为对于魔法使来说是必须的乘坐物。 迷你八卦炉是她最大的武器，同时也是她进行魔法实验和料理的必需品。 [ 史纪 ] 角色能力 魔理沙拥有 使用魔法程度的能力 ，早期也被描述为 操纵魔法程度的能力 和 主要拥有使用魔法程度的能力 。 她擅长使用光与热的魔法，注重魔法的火力。 不过，魔理沙是属水的。 [ 凭依华 ] 她的魔法虽然是只有破坏物品程度的魔法，但是单纯论破坏力的话在人类之中是非常强大的。 她使用的魔法弱点很少，无论对人还是妖怪都很有效，因此她也常常接受降伏妖怪的委托。 在符卡规则之下，魔理沙那华丽的魔法被发挥得淋漓尽致。 尽管是华丽无比的魔法，背后却是朴实无华的启动方式。 她会在 魔法森林 中采集原料、加工，然后以不同的方式处理，寻找可以启动魔法的方法。 [ 史纪 ] 她能使用依靠漂浮在周围的星星碎片的符卡，大气中的星之成分会左右其强度。 [ 魔理沙的魔法书 ] 她能使用火系魔法 发出火焰旋涡 [ GoM ] ，还 擅长让树叶随风飘舞的木系魔法 [ GoM ] 。 此外，她还会用召唤温泉地脉的魔法和炼丹魔法，前者一度是她一段时间使用的最大魔法。 [ 妖妖梦Manual ] 她自认为可以靠魔法变身，只是有风险。 [ 交叉评论 ] 她对周围的力量变化很敏感。 她曾感觉出 纯狐 力量并非魔力。 [ 绀珠传 ] 她还曾感觉到 矢田寺成美 背后魔力涌出来的方式很不寻常。 [ 天空璋 ] 她拥有速度和力量，但却不擅长体术。移动很迅速，无奈招数迟钝。她通过在魔法上增加必要以上的力量来弥补。 [ 萃梦想设定文档 ] 虽说主要使用魔法，但是她力气很大，甚至曾 抱着石头婴儿与 牛崎润美 战斗，并被润美夸赞臂力惊人 [ 鬼形兽 ] 。 她体力远超一般人，是鬼形兽的三个自机中唯一一个 打败 骊驹早鬼 后没有感觉到疲惫的角色 [ 鬼形兽 ] 。 她能闻出血的味道等“战场的味道”。 [ 茨歌仙 ] 外貌特征 魔理沙身高是 十多岁的样子，属于身高较低组 [ ZUN的邮件答复 ] 。 魔理沙有着一头金色卷发，长度从及肩到及腰不等。在有的作品中，一侧用蝴蝶结扎着麻花辫，所用的蝴蝶结颜色及位置也因作品而异。头戴巨大的黑色三角巫师帽，帽子上有白色、紫色、粉色或红色（花映塚2P）大蝴蝶结，在兽王园中蝴蝶结外白内粉，形似猫耳，在大战争中帽子上还有黄色星星装饰，在星莲船中帽子上则饰有红色星星和扫帚形装饰。 她的瞳色在虹龙洞及之前的作品中为黄色，在黑市、兽王园中为红色。 她在大部分作品身穿黑色连衣裙，在红魔乡中外穿着白色全围裙，之后的作品除了地灵殿和星莲船都披着白色半围裙，在星莲船中披着米黄色围裙并有“M”形图案。衣领前有时会系着领结或者绳子。她裙子里穿白色衬衫，裙下穿着白色灯笼裤，冬天会穿一件黑色或白色的披肩。在天空璋中，戴着粉色的围巾和短手套。 深信着魔法使穿黑衣服的理由是因为黑衣服就算脏了也不容易看出来 [ 风神录角色设定 ] 。 由于这样的配色，她被称为“黑白”。 帽子中、裙子里似乎藏有什么东西。 [ 史纪 ] 她脚穿白色袜子，长短从短袜到四分之三袜（漫画中也穿过黑色过膝袜等其他袜子）。她有时不穿鞋，穿鞋时穿的鞋有黑色乐福鞋、黑色玛丽珍鞋、棕色靴子、黑色靴子。 以上的黑色在一些作品中其实是深蓝色，但是在描述中仍称为黑色。 [ 地灵殿设定文档 ] 她手里常拿着八卦炉，也会拿扫帚甚至魔杖（星莲船）。 她常常坐在扫帚上。 红魔乡 妖妖梦 永夜抄（禁咒的咏唱组） 永夜抄（单人） 花映塚 花映塚2P 风神录 地灵殿 星莲船 神灵庙 辉针城 绀珠传 天空璋 鬼形兽 虹龙洞&黑市 兽王园 锦上京 妖精大战争 萃梦想 绯想天&非想天则 心绮楼 深秘录&凭依华 刚欲异闻 求闻史纪 儚月抄 茨歌仙 铃奈庵 智灵奇传 人妖名鉴 人际关系 魔理沙 是家族中的独女， 她的父亲 是 人类村落 中大道具屋雾雨店的主人 [ 史纪 ] 。 在她与自己的家族断绝关系时，曾在雾雨家修行的 森近霖之助 为她制作了魔法道具迷你八卦炉。 [ 香霖堂6话 ] 魔理沙管霖之助叫“香霖”。 [ 香霖堂 ] 魔理沙也常会拜访霖之助开的旧道具店 香霖堂 ，就连霖之助 也不知道她是有事还是没事 [ 香霖堂1话 ] 。 霖之助偶尔也会提及关于她父亲的事情，但魔理沙会刻意回避。 魔理沙与 博丽神社 的巫女 博丽灵梦 私交甚密，和 茨木华扇 、 东风谷早苗 等都是博丽神社频繁的访客。 红雾异变过后，魔理沙常会去红魔馆的大图书馆偷书， 被 帕秋莉·诺蕾姬 称为“老鼠” [ 萃梦想 ] 。 尽管如此，帕秋莉也曾 在 间歇泉骚动 时为魔理沙提供支援 。 魔理沙的符卡 恋符「Non-Directional Laser」 抄了红魔乡里帕秋莉的非符攻击。 [ 幻想揭示板 ] 作为人类村落中的借书屋铃奈庵的店主女儿兼店员， 本居小铃 和魔理沙也很熟络。 [ 铃奈庵 ] 同为魔法森林的魔法使住客，魔理沙经常到 爱丽丝·玛格特洛依德 家串门。 由于同为收集家，两人的关系可说是水火不相容。 [ 妖妖梦设定文档 ] 尽管如此，两人也曾 一起解决 虚假月亮的异变 ，爱丽丝还曾 在 间歇泉骚动 时为魔理沙提供支援 。 魔理沙的符卡 光击「Shoot the Moon」 在光弹中加入魔法的做法参考了爱丽丝会爆炸的人偶。 [ GoM ] 魔理沙与独居在魔法森林中的地藏魔法使 矢田寺成美 关系也不错。 成美曾为魔理沙再次挑战 摩多罗隐岐奈 出谋划策。 [ 天空璋 ] 三头慧之子 目击了魔理沙过去哭着逃进魔法森林的场面。而魔理沙之前就知道有一只白色的山犬逃进了 魔法森林 里，并 经常在魔法森林里看见她 [ 兽王园 ] ，对她很熟悉。 魔理沙解决异变时，会遭遇到各式各样的妖怪和人类，有的时候会发生战斗，有的时候则会互相协助。 比如 河城荷取 曾在 东方地灵殿 里为其提供协助， 古明地恋 曾于 东方凭依华 中与其组队。 隐岐奈也十分欣赏魔理沙，有将魔理沙纳为部下的想法， 但被魔理沙拒绝 [ 东方天空璋 ] 。 由于魔理沙丰富的人际关系，在 二次创作 中，以魔理沙和他人的关系为基础展开的创作类型也十分兴盛。 登场剧情 在和本家雾雨店断绝关系后，她哭着逃进 魔法森林 ，在魔法森林经营起一家万事屋“ 雾雨魔法店 ”。 虽然经营着雾雨魔法店，但平日里她很少在店里， 一点也不像在营业 [ 史纪 ] 。 工作的内容包括 赌博预言、解决异变 [ 史纪 ] 、 降伏妖怪、宝物探索、恋爱成就、子孙繁荣 [ 文果真报 ] 。 成功的话她就拿下报酬，失败的话则一文不取。 作为魔法使，她会使用魔法 召唤自己家地下的温泉地脉 [ 妖妖梦Manual ] 和 照明 [ 永夜抄Manual ] 。 她喜欢使用华丽的、高火力的魔法，却不愿被别人看到她努力的样子。 她 喜欢星星，认为星空中自有浪漫 [ 三月精V ] 。 她的符卡及自机攻击多与宇宙、星和光有关。 她对于降妖除魔十分在行，特别是符卡规则成立后，她华丽的魔法在符卡中发挥到了极点。 [ 史纪 ] 她曾一度痴迷炼丹， 以炼成丹为目标 [ 妖妖梦Manual ] ，但 制作的丹药有些过于大了所以吞不下去 [ 妖妖梦设定文档 ] 。 她还有偷盗的副业，甚至曾开玩笑说自己 早就转职盗贼 [ 地灵殿 ] 。 她最爱偷窃书籍。 她的行动方针是堂堂正正地闯进去，大大方方地说“暂时借走”。 [ 史纪 ] 每当发生异变时，她总是热衷于前去解决。遇到各种活动，她也总是前去参加。 直到今天，魔理沙仍然在东方的作品中，作为 主角（英雄） 而活跃着。 在原作者ZUN的概念里，魔理沙与天赋型主角灵梦相对，是成功背后默默努力的类型。 官作出场记录
