桑尼米尔克是东方Project系列中的官方角色，首次登场于 东方三月精E ，担任三月精系列的主要角色。 她是日光 妖精 ，从不知道什么时候开始她就居住在了 幻想乡 当中。与 露娜切露德 、 斯塔萨菲雅 合称为 光之三妖精 。 她是 光之三妖精的头脑 [ 三月精S6话 ] ， 是三人之中头脑最机灵的，表情丰富又开朗，精神十足。不过她其实也是失败次数最多的那个。 [ 上海爱丽丝通信 ] 。 除三月精外，桑尼还作为Boss登场于 妖精大战争 中，在其他官方出版物中也时有出场。 生活状况 光之三妖精 一起住在 魔法森林 之中的“自亘古以来便矗立着的大树” [ 上海爱丽丝通信 ] 内 三妖精的家 中，后光之三妖精搬家到 位于博丽神社内外分界线 [ 三月精S ] 的 大水楢树 [ 三月精O ] 中。 桑尼貌似领导着光之三妖精进行恶作剧，但三人没什么配合。 [ 上海爱丽丝通信 ] 十分喜爱美味的食物，乃至于常常不想太多而潜入宴会中。 [ 史纪 ] 角色能力 桑尼拥有 折射光线程度的能力 。 她可以让人类看到虚像，也可以让人迷路。 [ 上海爱丽丝通信 ] 她也能隐藏在折射的背后，使别人难以发现自己。 [ 史纪 ] 她能够通过沐浴日光来治愈伤势，相比之下则比较怕雨。 [ 上海爱丽丝通信 ] 外貌特征 桑尼身高很低， 不到一米 [ 上海爱丽丝通信 ] 。
