八意永琳是东方Project系列中的官方角色，首次登场于 东方永夜抄 ，担任永夜抄的六面Boss。 她是月人， 也是神明 [ 秋例11对话 ] ，是 月之都 的月之贤者之一，被称为“月之头脑”。 “永琳”是她使用的名字之一，她还有另外一个在月之都使用的名字。 [ 儚月抄漫画2话 ] 在永夜抄之后，永琳在官方游戏的结局和官方出版物中时有出场。 生活状况 永琳曾作为月之贤者之一居住在 月之都 中。 她现在居住在 永远亭 中，经营着名为 “蓬莱药局” [ 文果真报 ] 的药店。 角色能力 永琳是个 天才。 她的知识即使在月球的天才药师家族八意家中也显得拔群。她的头脑远远凌驾于人类之上，亦因此会经常说一些令人难以理解的话。 [ 永夜抄设定文档 ] 她发明了能够凭借量子的特性判明读信人的数量的量子印，并且这种印至今也没有别人会做。 [ 儚月抄小说1话 ] 她还拥有 制作任何药物程度的能力 。 人类用的药和妖怪用的药她都能做出。即使是疑难杂症，她也能开出特效药。她做出的药世间评价是无副作用，有不错的效果。 [ 史纪 ] 因为具有一切药物的知识，所以毒和药什么的对永琳都无效。 [ 永夜抄 ] 比起 蓬莱山辉夜 ，她拥有压倒性的力量。 [ 永夜抄设定文档 ] 由于 服下了蓬莱之药 [ 永夜抄 ] 而不老不死，同时会 产生和人类同样的污秽 [ 儚月抄小说1话 ] 。具体效果参见 相关整理 。 不过，因为永琳是神明，是另一个层面的存在，所以吃什么药、吃没吃药对永琳来说都没有区别。 [ 秋例11对话 ] 外貌特征 永琳有着一头白色及膝长发，在脑后扎成麻花辫。头戴蓝色护士帽，有红十字和白色星座图案。 她有着黑色眼瞳，但在部分官作结局中的Alphes绘瞳色有红色、褐色、蓝色等其他颜色。 她上身穿着左蓝右红的短袖，领子和扣子为白色。
