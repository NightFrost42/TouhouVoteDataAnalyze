新作角色。半人半灵的园艺师，脑子缺根筋的二刀流高手 角色介绍 魂魄妖梦是东方Project系列中的官方角色，首次登场于 东方妖妖梦 ，担任妖妖梦的五面Boss和六面道中Boss。 她是人类和 幽灵 的混血，而且还是 半人半灵 ，担任 白玉楼 的 园艺师 ，并兼任 冥界 大小姐 西行寺幽幽子 的护卫和剑术指导。 ZUN在参加第二次（2003年）和第三次（2004年）东方人气投票时， 均将角色部分的三张票之一投给了妖梦 [ ZUN的人气投票记录 ] 。 由于年龄尚小，性格认真直率而不成熟， 脑子有些缺根筋 [ 鬼形兽设定文档 ] ， 经常被人戏弄 [ 永夜抄角色设定 ] 。 在妖妖梦之后，妖梦还作为自机登场于 东方萃梦想 、 东方永夜抄 、 东方绯想天 、 东方非想天则 、 东方神灵庙 、 东方鬼形兽 中，在官方出版物中也时有出场。 生活状况 作为 白玉楼 的园艺师，她大多数时间位于 冥界 ，主要负责庭院的巡逻和修剪工作，早晚都会各巡视一次庭院，有时也会外出购物或处理冥界事务。业余时间则会在白玉楼修行剑术。 因为 白玉楼的庭院十分广阔，并且栽种着大量的樱花 [ 妖妖梦角色设定 ] ，她 每日的工作十分繁重 [ 神灵庙角色设定 ] 。 本来她还是幽幽子的剑术指导，但现在仅仅过着园丁待遇的日子。 [ 永夜抄Manual ] 另外，由于幽幽子表现出的 吃货属性 [ 永夜抄 ] ， 在 二次创作 中也常常有着妖梦作为白玉楼厨师负责幽幽子餐食的设定 。 角色能力 妖梦拥有 使用剑术程度的能力 。 她是 二刀流 的高手，使用妖怪锻造的长刀 楼观剑 和魂魄家的家传短剑 白楼剑 。 楼观剑 拥有一挥便可杀伤十只幽灵的威力 [ 萃梦想设定文档 ] ，自称 无法斩断的东西几乎不存在 [ 妖妖梦 ] 。 然而 幽灵通常是无法杀伤的 [ 史纪 ] ，所以真实性待考。 白楼剑 可以斩断人类的迷惘 [ 萃梦想设定文档 ] ， 只有魂魄家的人可以使用，原因不明 [ 史纪 ] 。 对幽灵使用会成佛；对人使用一说 因为锋利度糟糕而只会有点痛 [ 史纪 ] ，一说 可以斩断活着的人类 [ 萃梦想设定文档 ] 。 说起爆发力和集中力，据说没有人能超越身为二刀流的她。 [ 上海爱丽丝通信 ] 基本上 从敌人、灵体到弹幕甚至人的烦恼都可以斩断 [ 萃梦想设定文档 ] 。 半灵部分 可以按照自身的意志活动， 并且能放出一定规模的弹幕 [ GoM ] 。 总之 体术和妖术都是半吊子但很优秀的程度，保持了平衡 [ 萃梦想设定文档 ] 。 她还能够 以极高的速度进行移动，但似乎只能做短距离直线移动，且移动前必须进行准备 [ 文花帖 ] 。 超高速移动时连速度很快的 射命丸文 也无法用眼睛捕捉其身影 [ 文花帖 ] 。 在进行白玉楼庭院的修剪工作时似乎也使用着二刀流和超高速移动的能力。 外貌特征 妖梦的 外表是十多岁的少女，属于身高较低的角色 [ ZUN的邮件答复 ] 。 她有着一头银白色及肩短发，梳成娃娃头，戴着黑色的丝带发饰。 她的眼瞳在妖妖梦中或 过度注视满月 [ 永夜抄 ] 后为红色，在萃梦想、永夜抄、花映塚、神灵庙、鬼形兽中为蓝色，在绯想天、非想天则、人妖名鉴中为青绿色。
她上身穿着白色衬衫，在妖妖梦、花映塚中为长袖，在其他作品中为短袖。衬衫外面套着 青绿色 短褂，系着黑色领结。短褂左胸口有口袋和白色的幽灵状图案（鬼形兽中改为眼睛状）。 除史纪、绯想天、非想天则、鬼形兽外，她背后背着两把黑色剑鞘。
她下身穿着 又短又宽的 [ 香霖堂 ] 青绿色裙子，裙摆边沿分布着白色幽灵状图案。裙下穿着白色灯笼裤。 她脚穿白色短袜和黑色玛丽珍鞋，袜子在妖妖梦、史纪中有白色褶边，在人妖名鉴中为三折袜，在其他作品中为泡泡袜。 她在永夜抄中手持长刀「楼观剑」在花映塚中手持短剑「白楼剑」，在史纪、鬼形兽中双手持剑，在萃梦想中右手持剑，在绯想天、非想天则中手持白楼剑鞘，左腰挂着楼观剑鞘。 楼观剑有着 不适合普通人使用的长度 [ 史纪 ] ，在妖妖梦、绯想天、非想天则中刀柄上有樱花图案，在妖妖梦、神灵庙中绑着白色布条，在鬼形兽中绑着蓝色布条，末端有一撮幽灵般的蓬松白毛。在妖妖梦、绯想天、非想天则、神灵庙、人妖名鉴中剑鞘上绑着一朵 秋樱（大波斯菊） [ 紫香花 ] 。剑格在鬼形兽中为金色，在其他作品中为黑色（在史纪中无剑格）。 白楼剑在人妖名鉴中剑柄绑着白色布条，在鬼形兽中绑着蓝色布条，在妖妖梦中绑着两朵白色小花。在妖妖梦、花映塚、史纪、人妖名鉴中有剑格，在妖妖梦、花映塚中为黑色，在人妖名鉴中为金色。 作为半人半灵的标志，幽灵半身有着白色的灵体外形，通常浮在人类半身旁边。 妖梦的2P色，将青绿色改为绿色，将粉红改为蓝色。 妖妖梦 永夜抄（幽冥的居民组） 永夜抄（单人） 花映塚 花映塚2P 神灵庙 鬼形兽 萃梦想
