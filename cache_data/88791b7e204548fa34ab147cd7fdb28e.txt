红美铃是东方Project系列中的官方角色，首次登场于 东方红魔乡 ，担任红魔乡的三面Boss。 她是 中国妖怪 [ C76对话 ] ，身穿中国风的服装，使用中国武术，担任着 红魔馆 的门卫。 虽然她的种族为妖怪， 但她的外表和谈吐与人类没有任何区别 [ 东方求闻史纪 ] 。 性格开朗率直，有些热血。偶尔会偷懒和发牢骚。 在红魔乡之后，她也作为自机登场于格斗作 东方萃梦想 和 东方非想天则 中。 生活状况 因为擅长体技 [ 上海爱丽丝通信 ] ，美铃被任命为 红魔馆 门卫。 红魔馆很少接待客人，所以她经常会阻拦不速之客 [ 非想天则 ] 。 同时她也负责 管理红魔馆的花坛 [ 文花帖（书籍） ] ，被认为是 花坛的看护人 [ GoM ] 。 看门时，她会 打太极拳、睡觉 [ 东方求闻史纪 ] 或 看热血漫画 [ 非想天则 ] 。 因为她由红魔馆提供饮食 [ 史纪 ] ，所以不会主动袭击人类。 不仅如此，因为职务关系，她和人类接触率很高。经常有人和她搭话。 [ 史纪 ] 来拜访的人有时会 向她发起挑战，测试自己的身手 [ 史纪 ] 。 角色能力 美铃拥有 运用气程度的能力 ，尤其擅长配合“气”使用的武术。 武艺高强 [ 幻想揭示板 ] ， 行动也很轻巧 [ 史纪 ] 。 虽然她的种族是妖怪， 但是她很少使用妖术 [ 史纪 ] 。 是万能型的妖怪，没有特别的弱点，因此虽然不算强但面对人类没有死角。 [ 史纪 ] 除此之外，作为花坛看护人，她也使用过 以开花为印象的弹幕（ 华符「芳华绚烂」 ） [ GoM ] 。 外貌特征 美铃属于 身长高 [ ZUN的邮件答复 ] 的角色， 外表和人类没有任何区别 [ 史纪 ] 。 她有着一头红色及腰长发，两侧的鬓发扎成麻花，末梢系着黑色蝴蝶结，头戴绿色帽子，帽子前面有一个写着“龍”字的金属星形物。 她有着蓝色的眼瞳。 她上身穿着军绿色马甲和白色短袖衬衣，马甲上边缘有白色回纹，门襟为白色，扣子为一字扣，领口有黑色丝绳系成的蝴蝶结。 她下身穿着长度到小腿的军绿色长裙，在人妖名鉴中右侧开衩，开衩处旁边有浅绿色回纹；在除人妖名鉴外的其他作品中在左侧开衩，开衩处为白色，在红魔乡中有绿色回纹，在萃梦想、非想天则中有灰色回纹。 她在史纪中穿着白色短袜和黑色鞋子，在人妖名鉴中穿着黑色船鞋。 红魔乡 萃梦想 非想天则 求闻史纪 人妖名鉴 人际关系 十六夜咲夜 是美铃的上司，美铃睡觉时会为了让她好好工作 把她打醒 [ 东方非想天则 ] 。
