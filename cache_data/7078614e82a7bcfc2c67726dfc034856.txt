东方红魔乡 ，担任红魔乡的EX面Boss。 她是 蕾米莉亚·斯卡蕾特 的妹妹，与蕾米莉亚一样是居住在 红魔馆 的 吸血鬼 。 平常并不太易怒，但精神有些问题。 在红魔乡之后，芙兰朵露还作为自机&敌机登场于 东方刚欲异闻 中，在官方出版物中也偶有出场。 生活状况 芙兰朵露生活在 红魔馆 的地下室里， 在 红雾异变 前的495年间从未外出过 [ 红魔乡 ] ，如今也 极少外出红魔馆 [ 史纪 ] 。 芙兰朵露的年龄也约为495岁 [ 红魔乡设定文档 ] ，因此红雾异变前，芙兰朵露几乎是从出生起就没有外出过。 虽然红魔馆的其他人因为芙兰朵露拥有危险的能力和 精神有些问题 [ 红魔乡设定文档 ] 而限制她离开红魔馆，但其实 芙兰朵露是觉得住着挺舒服才老实地待在红魔馆里 [ 智灵奇传 ] 。 角色能力 芙兰朵露拥有 将一切悉数破坏程度的能力 。 她能够将“目”（事物最重要的部分）移动到自己的手中，并且一握拳就能将手中的“目”破坏，甚至不用触摸到事物本体就能破坏事物。 [ 史纪 ] 作为 吸血鬼 ，芙兰朵露还 拥有吸血鬼超越常人的身体能力 [ 史纪 ] 。 外貌特征 芙兰朵露属于 身高低 [ ZUN的邮件答复 ] 的角色， 身高和姐姐蕾米莉亚差不多 [ 史纪 ] 。 她有着一头浅黄色齐肩短发，略卷，左侧扎着更长一些的侧马尾。头戴形似门把手套的睡帽，装饰着红色缎带，左侧系成蝴蝶结，侧马尾从蝴蝶结下帽子的洞中穿出。 她有着红色眼瞳。 她上身穿着红色马甲，领子打着数层白色褶子，内穿白色短袖。胸前系着黄色领巾。在求闻史纪和人妖名鉴中，手腕上有红色腕带。 她背后长着 一对散发着七色不可思议之光的翼，有着其它生物、妖怪、幽灵无法想象的奇怪扭曲形状 [ 史纪 ] 。在红魔乡中，两翼上各挂着8个彩色结晶，晶体由里到外的颜色顺序为：天蓝，蓝，紫，粉，橙，黄，浅绿和天蓝。在其他作品中，晶体数量均少于8对，且颜色也有差异。 她下身穿着红色短裙，有白色褶边。在求闻史纪中则穿超短裙。在刚欲异闻中则是在里面穿白色带褶边短裙，外面在两侧披着红色布料，布料在下端有一圈白线，还在腰间系着一条粉色带子，在身后垂下。 她穿着白色短袜和白色脚环，在求闻史纪中脚穿短靴，在人妖名鉴中脚穿棕色玛丽珍鞋。 她在红魔乡、求闻史纪、刚欲异闻中手持一根S形的棒子，两端像扑克牌中的黑桃，在刚欲异闻中前端黑桃后面系着红色蝴蝶结。 这根棒子 不是 禁忌「Laevatein」 中芙兰朵露所使用的剑 ，莱瓦汀是 一把大得离谱的炎之剑 [ GoM ] 。 红魔乡 刚欲异闻 求闻史纪 人妖名鉴 人际关系 芙兰朵露是 蕾米莉亚·斯卡蕾特 即使是红魔馆中，和她友好的妖怪也不多，常常被孤立。很少见到她和姐姐蕾米莉亚在一起。 这么说来，她身上还有很多谜。她的破坏能力，据说不管有没有触摸到那个事物都能将其破坏。要和她一起玩的话，物品都会被破坏得一干二净，没人会愿意这样吧。 所有的事物都有最重要的部分称之为“目”，要破坏事物的话，只要加力于它，就能轻而易举地将其破坏。她能够将“目”移动到自己的手中，并将手中的“目”破坏
