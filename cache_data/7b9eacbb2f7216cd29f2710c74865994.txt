，担任妖妖梦的EX面boss和PH面道中boss。 蓝是凭依在九尾妖狐上的式神，而这只妖狐却是另一只妖怪。这只妖狐有着强大的魔力，达到了妖兽的最高峰。虽然凭依着最强妖兽，她却性格温和，不会为了自己而争斗。 此外，她还保留着一只名为“蓝”的狐狸动物灵的特征与记忆。 1 [ 兽王园 ] 她被 八云紫 用于负责各种杂事。同时，她也能使役包括 橙 在内的多个式神和创造新的式神。 她的原型是日本传说中的“玉藻前”， ZUN甚至暗示她就是这只祸乱三国的九尾妖狐 [ 妖妖梦 ] 。 在妖妖梦之后，长期作为八云紫的符卡登场，还作为自机登场于 东方兽王园 中，也在官方出版物中时有出场。 生活状况 动物灵蓝曾在 畜生界 活动，后 因受不了畜生的思想而离开畜生界前往地上 [ 兽王园 ] ，但是 仍然关心着畜生界，当畜生界有事相拜托时也会回去帮忙 [ 香霖堂 ] 。 目前八云蓝和 八云紫 同住在幻想乡 艮之境 [ 妖妖梦 ] 的 住所 里。 作为紫的式神，她常常代替紫在幻想乡行动，负责各种杂事和完成紫的命令。 紫把一切杂事都交给她。日子过得苦不堪言。 [ 永夜抄 ] 她非常喜欢油炸豆腐，会去 人类村落 里买油炸豆腐。 她喜欢摆弄电子计算机。 [ 香霖堂 ] 角色能力 蓝拥有 使役式神程度的能力 ，能役使包括 橙 在内的多个式神。 但是，蓝不会把工作派给不够成熟的橙，只能亲自执行任务。 蓝给橙编的程序有很多bug，使橙总是一下就不见了，紫对此很不满意 [ 地灵殿 ] 。 此外，蓝也能 自己做出式神 [ 妖妖梦 ] 。 蓝还拥有着相当高的智力，尤其擅长数学， 能瞬间完成人类所无法想象程度的计算 [ 史纪 ] 。 曾为了打发时间而计算出三途河的河宽 [ 文花帖（书籍） ] 。 因此， 比起构思新的概念，她更倾向于分析求解现有的事物 [ 史纪 ] 。她 对于文字的处理能力也差得远 [ 儚月抄 ] 。 因为这些不足，身为主人的紫对蓝有着更高的期待，会 引导八云蓝自己主动思考而不是被动执行命令 [ 儚月抄 ] 。 外貌特征 蓝属于 身长较高 [ ZUN的邮件答复 ] 的角色。 她有着一头黄色齐脖短发，头上长着一对狐狸耳朵，戴着一顶白色帽子遮住了耳朵。帽子上贴着许多黄色符纸，在兽王园中上面有红色图案。帽子边缘有荷叶边，两侧顶端各有一个绒球和穗子，绒球和穗子在妖妖梦中为淡蓝色，在人妖名鉴和兽王园中为白色。 她有着黄色的眼瞳。 她身穿白色长袖长袍，身前披着长到小腿的蓝色布料，有着白色花纹，整体形似道袍，在人妖名鉴中，布料两角有绿色绒球和穗子。袖子边缘和衣领为紫色，并打着白色褶子。 在妖妖梦中，她脚穿白色短袜，脚腕上有蓝色脚环，腿环打着白色褶子。在求闻史纪和人妖名鉴中，她脚穿白色短靴。 她身后有九条金色尾巴，尾巴尖为白色。 她平时总是把手对插进袖子里，不过在兽王园中，她挽起右臂的袖子，右手拿着一张猫形符纸，上面有红色图案， 三途河的河宽计算成功 摄影协助·六羽田共 ○月○日，数字的魔术师算出了三途河的河宽，让相关的人都非常吃惊。说到三途河，就是存在于这个世界和那个世界之间的一条河。这条河的有名之处就在于其河宽不是固定的，一旦满足一定的条件河宽也就会随之发生极大的变化。因此，要计算河宽几乎被认为是不可能的。 数字的魔术师为了打发时间…… 完成了这个伟业的是八云蓝（式神）。平常就对数字方面的东西很擅长，就算是复杂的算式也能够在瞬间算出答案，经常让周围的人觉得惊奇。当问到她为什么这次想到要去计算三途河的河宽的时候，她给出的却是“打发时间”这样一个简简单单的答案。 虽然三途河的河宽很让人在意，不过因为理解那个算式需要一定程度的专业知识，所以该报道就不对算式进行详细说明了。对此有兴趣的人推荐亲自去向当事者本人询问。关于概要如下所述。 「三途河不是普通的河而是流淌着过去的河。因此其长度和眼睛所能看到的长度不同，要算出其距离的话必须要对那个死者的历史有概念才行。河宽根据每个人的过去有可能在一瞬间完结，也有可能过了很久都无法渡过。而这个因子虽然人类可能不清楚，但其实就是给摆渡人的渡船钱。给的钱越多，距离就会缩得越短。二者间呈现完美的反比例关系，摆渡费接近零时河宽会趋于无限大，而随着摆渡费增加河宽则趋近于0」
