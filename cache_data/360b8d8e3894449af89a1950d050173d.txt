兼任红魔馆的清洁工 [ 红魔乡设定文档 ] ， 是馆中 唯一的人类 [ 史纪 ] 。 “十六夜咲夜”这个名字是蕾米莉亚赋予她的。 [ ZUN的邮件答复 ] 至于她的本名，ZUN是有过相关考虑的，包含这一点在内，咲夜这个角色充满了谜团，ZUN对她的身份背景做了更加完整神秘的设定，似乎 连她在 东方妖妖梦 中使用的Bomb都被赋予了特殊的含义 [ ZUN的邮件答复 ] ，但这一切至今仍未公布。 她行事潇洒从容， 头脑敏锐 [ 妖妖梦设定文档 ] ， 平时总是镇定自若的样子 [ 虹龙洞设定文档 ] ， 但也有着让人摸不清在想什么、脱线的一面 [ 萃梦想设定文档 ] ， 有点天然 [ 虹龙洞设定文档 ] 。 在红魔乡之后，她也常常于正作和格斗作中作为可选自机出场，在官方漫画与小说中也屡有戏份。 生活状况 身为女仆长，咲夜大部分时间都待在 红魔馆 中，负责 打扫卫生、指派任务、做料理、服侍大小姐 [ 妖妖梦角色设定 ] 等，偶尔会外出进行采购。 尽管她需要完成各种工作以及满足蕾米莉亚的任性需求，还 没有工资 [ 妖妖梦角色设定 ] ，但她本人对工作却 没有任何不满等阴暗的情绪 [ 永夜抄Manual ] ，甚至会 陪蕾米莉亚打发闲暇 [ 三月精S ] 。 凭借自己优秀的个人素质，以及 操纵时间的能力 ，她总是能完美地完成所有工作。 至于为什么咲夜会在红魔馆里工作、以及她之前是什么样的人，全都充满谜团。 有一种说法称，她也许原本是 外面世界 或是其他世界的吸血鬼猎人，追赶着吸血鬼来到了红魔馆，却反被击败。此外，虽然还有其他的诸如人造人说、丧尸说、月人说、单纯只是怪人说等各种各样的说法，但是其中最有说服力的还是吸血鬼猎人说。 [ 东方求闻史纪 ] 此外，她也时常出席 博丽神社 的宴会并经常负责为宴会准备料理。 为了满足蕾米莉亚、 帕秋莉·诺蕾姬 的要求或者发现异变后，她也会为此外出在幻想乡内奔波。 她怕吃热食。 [ 妖妖梦设定文档 ] 角色能力 咲夜拥有 操纵时间程度的能力 和 停止时间程度的能力 。 由于 能够操作时间就能够操作空间 [ 幻想揭示板 ] ，咲夜也能操作空间，甚至被 帕秋莉·诺蕾姬 评价为 喜爱摆弄空间 [ 红魔乡 ] 。 操纵时间的能力对人类而言是 最高等级的强大能力 [ 史纪 ] ，并不是一般人能通过修行取得的。 咲夜自称，她的时间停止能力，其实为 超高速的移动能力 [ 茨歌仙35话 ] 。 通过自如地运用这份能力，咲夜可以做到瞬间移动自己和物体、 在时间停止中扫除 [ 红魔乡设定文档 ] 、 让竹子立刻开花 [ 红魔乡 ] 、 拓宽红魔馆的内部空间 [ 幻想揭示板 ] 等等，当然也可用于战斗。 咲夜还是使用飞刀的高手， 至于其精准度，如果让妖精女仆头顶苹果站在十二丈外，据说她投出的小刀可以正中妖精女仆的 额头 [ 史纪 ] 。 小刀用尽后会停止时间进行回收，投掷完了也会进行回收。 [ 妖妖梦设定文档 ] 咲夜 对料理也非常拿手 [ 妖妖梦设定文档 ] 。高超的厨艺与红魔馆的豪华食材相搭配（特别是肉类食材）， 使她在宴会上有着很高的人气 [ 茨歌仙33话 ] 。 咲夜是一位 表面上的魔术师 ，因为 她的魔术不需要手法和技术，只是单纯地操纵了时间 [ 幻想揭示板 ] 。 说白了就是在时间暂停期间移动物品的位置等等，在常人眼中看起来就像是凭空变出来的一样。 稗田阿求 却认为这隔空取物般的魔术 不是单纯操控时间而来的 [ 史纪 ] 。 此外，她 也擅长体术，不如说比起弹幕似的可疑的术她更擅长体术 [ 萃梦想设定文档 ] 。 她没有弱点。 [ 妖妖梦设定文档 ] 外貌特征 咲夜的外表是 10~20岁左右 [ 红魔乡设定文档 ] 的少女， 属于身长高的角色 [ ZUN的邮件答复 ] 。实际年龄成谜， 稗田阿求 认为 她虽然自称不到20岁，但行事沉稳得像数百年前的人 [ 史纪 ] 。 她有着一头银灰色及肩短发，除妖妖梦外两侧的鬓发扎成麻花并在末梢系着绿色蝴蝶结，除红魔乡及肩外鬓发及胸。她头上戴着喀秋莎头饰。 她的眼瞳在红魔乡中为红色，在妖妖梦中为灰色，在后续作品中为蓝色。 她身穿以深蓝、白两色为主色调的女仆装，包括衬衫、深蓝色上衣、带白色花边的深蓝色及膝裙和系在腰上的白色围裙，在妖妖梦中衬衫为蓝色，在其他作品中衬衫为白色。在妖妖梦中，她还戴着棕红色围巾；在红魔乡中，右袖边缘有“Red Magic”的刺绣；在花映塚中，围裙摆上印有罗马数字，形似时钟。 她在妖妖梦、永夜抄中脚穿白色短袜，在辉针城、虹龙洞、人妖名鉴中穿着黑色丝袜；在妖妖梦中穿着棕色皮鞋，在永夜抄中穿着深蓝色皮鞋，在辉针城、虹龙洞中穿着棕色短靴，在人妖名鉴中穿着黑色短靴。 除妖妖梦、萃梦想、人妖名鉴外，她手持银质小刀，在人妖名鉴中身后环绕着一圈银质小刀，在永夜抄中还手持怀表。
