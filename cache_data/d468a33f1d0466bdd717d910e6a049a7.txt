八坂神奈子（官方角色，新作） - 新作角色。山与湖之神，热衷技术革新的建御名方神 角色介绍 八坂神奈子是东方Project系列中的官方角色，首次登场于 东方风神录 ，担任风神录的六面Boss和EX面道中Boss。 神奈子是来自 外面世界 的大和神，是 守矢神社 名义上祭祀的大国主的儿子“建御名方神”，是战神 [ 儚月抄 ] ，实际上又 是风雨之神，现在也是山之神明 [ 口授 ] ，还被当作农业之神祭祀，并 打算慢慢转型为技术革新之神 [ 口授 ] 。 虽然是风神，但她很少自称风神。 [ 茨歌仙 ] 尽管 被设定为蛇的形象 [ 风神录采访 ] ，背着蛇状的注连绳，会 使用蛇作为弹幕 [ Ground Serpent ] ，也曾被称为 大蛇 [ 口授 ] 、 蛇神 [ 刚欲异闻 ] ，但是神奈子并没有本体为蛇或身为蛇神的设定。 她的 性格非常专横并且以自我为中心 [ 口授 ] ，而看起来十分可靠。 她大大咧咧的， 有着爽快的性格，因此与魔理沙相当地合得来 [ 风神录 ] 。 ZUN在采访中称将神奈子设定成了 蛇一般的性格，即十分狡猾，擅长欺骗人，并有二枚舌 1 的印象 [ 风神录采访 ] 。 村纱水蜜 觉得神奈子 假惺惺的 [ 刚欲异闻 ] 。 她特别喜好技术革新，有不重视传统的倾向，不管什么样的信仰都有在收集。 [ 口授 ] 她似乎很容易生气，但是对礼物没有抵抗力。 [ 口授 ] 在风神录之后，神奈子还作为自机&敌机登场于 东方刚欲异闻 中，并多次为 东风谷早苗 提供支援，在早苗线结局中经常出场。在官方出版物中，作为守矢神社的代表时有出场。 生活状况 神奈子从 洩矢诹访子 手中夺得洩矢之国后，因未被接受、没有得到信仰心，而放弃王国的所有权并借来诹访子的力量君临了山里。 [ 风神录 ] 现在她居住在 守矢神社 ，是守矢神社的 “销售部” [ 风神录 ] ，主管对外交涉。 然而，神奈子 真正的住所不是在神社而是整个 山 [ 风神录 ] 。 她经营的营业活动相当丰富。 [ 口授 ] 她管理着 核聚变炉 ，若其运行出现异常会前往调查。 角色能力 神奈子拥有 创造乾程度的能力 。 使用这项能力，神奈子能掌管刮风下雨，所以可以带来丰收，因而被作为农业之神祭祀。 此外， 建造道路或是巨大的建筑物也是她的专长 [ 口授 ] 。 作为山神， 改变地形 [ 口授 ] 是神奈子的御神德。 外貌特征 神奈子 容貌非常庄严 [ 口授 ] 。 她有着一头向左右伸展、容量惊人的蓝紫色齐肩短发，脑后用注连绳系着一个发髻，绳上装饰着银杏、枫叶和醋栗，在刚欲异闻以外的作品中装饰在右侧，在刚欲异闻中在左侧。 她的瞳色在风神录、刚欲异闻中是接近茶色的红色，在人妖名鉴中是蓝色。 她上身穿着红色的短袖，袖口有用金属的袖口卡住。短袖之下还穿了一件白色的宽松长袖。她胸前佩戴着一个黑色镜子，在白色长袖的袖口、腰上以及脚腕上都戴着注连绳环。 她背后背着巨大的注连绳圈，其上挂着许多纸垂。 她下身穿着深红色长裙，裙摆上有梅花图案。 她赤脚踏着一双草鞋。 在人妖名鉴中和战斗时，她背后还会装备许多御柱。 风神录 刚欲异闻 求闻口授 人妖名鉴 人际关系 神奈子曾经入侵 洩矢诹访子 所拥有的洩矢之国，两人是敌对关系。 后来在神奈子夺取王国后两人共同统治王国，诹访子还借给神奈子力量，两人关系变得融洽。 现在两人早已成为 利害关系一致，而且实际上也很要好 [ 风神录 ] 的朋友。 神奈子所依仗的神德和信仰心基本都是源于诹访子的力量。 [ 风神录 ] 神奈子作为山神是借来了诹访子的力量。 神奈子作为山神的御神德是改变地形，属于诹访子的能力范围。 神奈子作为农业之神被祭祀，而诹访子同样拥有使土地丰收或成为不毛之地的能力。 东风谷早苗 是 神奈子的巫女 [ 风神录 ] 。 神奈子赐予 灵乌路空 八咫乌的力量，将阿空驯养，并作为上司让阿空维持 核聚变炉 的正常运转。 神奈子代表 守矢神社 和由 饭纲丸龙 等大天狗与 天魔 所领导的 天狗 社会 缔结和平条约 [ DS ] ，相互利用。 神奈子少量地传授科学技术给 河童 来征收生活费。 [ 口授 ] 不过，整体而言 神奈子在幻想乡里没什么朋友，甚至有些被人们讨厌 [ 智灵奇传 ] 。 登场剧情 官方游戏部分 在 东方风神录 中，将 守矢神社 搬进幻想乡 妖怪之山 并在山上收集信仰，试图利用 博丽神社 却被自机找上门打败，但仍争取到建造分社，后又试图阻止自机打扰 洩矢诹访子 。 在 东方地灵殿 中，为了进行核聚变前往 灼热地狱遗址 ，将八咫乌的力量给予了物色的地狱鸦 灵乌路空 并驯养了她。
