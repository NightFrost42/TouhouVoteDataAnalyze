藤原妹红是东方Project系列中的官方角色，首次登场于 东方永夜抄 ，担任永夜抄的EX面Boss。 她原是贵族家的女儿，因服用 蓬莱山辉夜 留下的蓬莱之药而成为不老不死的人类。在再次遇到辉夜后开始了与她无尽的厮杀日常。 在永夜抄之后，妹红还作为自机在 东方深秘录 和 东方凭依华 出场，在官方出版物中也时有出场。 生活状况 妹红会护送在 迷途竹林 迷路的人回到 人类村落 。如果拜托她担任在迷途竹林的护卫一定会答应，并 喜欢听拜托者讲家族的事，但自己沉默寡言 [ 史纪 ] 。 不会迷路的原因是 和竹林里的妖精混得很熟 [ 儚月抄 ] 。 这样做的原因仅仅是 不愿意看到迷路之人的尸骨 [ 儚月抄 ] 。 她 从得到的感谢中获得生存下去的动力 [ 儚月抄 ] 。 她居住在 隐藏于迷途竹林的小家中 [ 儚月抄 ] 。 从 约四百年前 [ 儚月抄 ] 开始，与 蓬莱山辉夜 厮杀成为她的日常，她们在相互死斗中找到乐趣。 她会到 永远亭 里赏月，但在不和里面的人玩。 [ 深秘录 ] 角色能力 妹红自从服用蓬莱之药后，就拥有了 不老不死程度的能力 。 人类喝了蓬莱之药后， 身体就不会朽坏，可以永远地活着 [ 儚月抄 ] ，但 不被允许死去，也不能成仙，再也不能像一个人类一样和其它人类生活 [ 儚月抄 ] 。 如果灵魂与肉体分离，那么肉体会立即毁灭，而灵魂会在喜欢的地方制造生长出新的肉体。 [ 永夜抄 ] 身体能迅速恢复任何程度的损伤，即使是重伤也能在几天内复原。 以此为基础， 她能以削减自己的体力发出的舍身攻击来进攻 [ 东方凭依华 ] 。 不过，虽然不会死亡，她仍然会饿、会痛。 因此， 西行寺幽幽子 称她为“蓬莱人” [ 永夜抄 ] 。 由于活了很久，她身上有各种妖术，据说能和妖怪势均力敌。 [ 史纪 ] 宫出口瑞灵 认为妹红的身份性质为妖怪，且瑞灵吸干了妹红的力量并导致妹红失去意识，这与瑞灵袭击妖怪时的特征一致。 她能够在某种程度上操纵火焰，有许多以不死鸟、火焰为主题的符卡。 外貌特征 妹红有着一头白色及踝长发。头发扎着许多蝴蝶结，脑后扎着一个大蝴蝶结，蝴蝶结均为红白相间的颜色。 她的眼瞳为红色。
