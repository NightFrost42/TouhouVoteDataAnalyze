新作角色。成为仙人的茨木童子，常帮助和指导博丽灵梦工作 角色介绍 茨木华扇是东方Project系列中的官方角色，通称“茨华仙”，首次登场于 东方茨歌仙 ，担任茨歌仙的主要角色。 她是 仙人 ，在 东方茨歌仙第四十九话下 中明确揭示了她是 鬼 的事实。她自称成为仙人的目的 “只是想要接近人类” [ 茨歌仙18话 ] 。 ZUN在 多处 [ 茨歌仙 、 深秘录 、 凭依华 ] 暗示华扇很可能是 幻想乡 的 贤者 ，但是始终 没有挑明华扇究竟是不是贤者 。 她 言谈举止柔缓和婉，对人类和妖怪都温柔对待 [ 香霖堂 ] ，但 非常爱说教 [ 茨歌仙1话 ] 。 在茨歌仙初登场之后，华扇又作为自机登场于格斗作 东方深秘录 和 东方凭依华 中，在其他官方出版物中也时有出场。 生活状况 尽管华扇 被认为住在 妖怪之山 中 [ 茨歌仙1话 ] （她也会如此自称） [ 茨歌仙42话 ] ，但严格来说她居住于 自己仙界 内的茨华仙邸。 她在仙界里饲养着自己的宠物们，并且会派它们办事，乃至在战斗中役使它们。 经常到 博丽神社 指导与帮助 博丽灵梦 的工作。 厌恶怨灵，常常毫无缘由地将怨灵捏爆。 尽管身为仙人，却放出接近于常人的气场。 [ 凭依华 ] 不过， 森近霖之助 却觉得她 散发着绝非善类的气场 [ 香霖堂 ] 。 角色能力 华扇拥有 引导动物程度的能力 。 她能听懂动物的话，动物也能听懂她的话。 [ 茨歌仙18话 ] 她饲养了很多珍禽异兽作为宠物。 不擅长对待鱼。 [ 茨歌仙22话 ] 丰聪耳神子 能从华扇身上感受到远超自己的强大力量。 [ 茨歌仙18话 ] 华扇有一定的感知与操纵结界和间隙的能力。 华扇曾 以贤者××××之名在 博丽大结界 上打开一个口子 。 [ 东方深秘录 ] 华扇能感受到结界的扭曲，并能通过念咒打开间隙。 [ 茨歌仙 ] 她被封印的右臂 茨木童子之臂 中寄宿着 鬼 的邪气，能够幻化出形貌。 [ 茨歌仙50话 ] 设定中也将其称作影华扇、影之茨华仙。 [ 茨歌仙设定资料图 ] 茨木童子之臂能够独立思考、行动、交谈。 茨木童子之臂的实力相当不俗，拥有鬼的力量，一度压制了 博丽灵梦 。 [ 茨歌仙49话 ] 华扇拥有的酒器“茨木的百药枡”（简称“百药枡”），把酒倒入其中后喝下去可以治疗疾病和伤势，还可以使健康人获得怪力，但代价是性格在一段时间内会变得和鬼一样，并且有伤病的人在治疗后身体本身也会逐渐变成鬼。 [ 茨歌仙16话 ] 外貌特征 华扇有着一头过肩粉色短发，头上用两个白色布套包成双丸子头以隐藏作为鬼特征的角。 她有着粉色的眼瞳。 她上身穿着白色短袖，袖口用红线系住，边缘为很长的锯齿状。 这可能是ZUN在设计时有意加入的“带刺蔷薇”的意象的体现 。 [ 人妖名鉴 ] 她身前披着深红色及膝挂巾，边缘为金色，上面有金色麦穗状图案，胸口有一朵粉色牡丹花，在深秘录、凭依华中还带有萼片。 当她身为完全体的鬼形态时，上身的衣物变为无袖，与身前的挂巾均为黑色。 在人妖名鉴中，鬼华扇身着的衣物为蓝色。 她下身穿着绿色及膝裙，边缘有一圈虚线，在深秘录、凭依华中为深红色，在人妖名鉴中为深绿色。 她脚穿船鞋，脚腕系着腕带，在深秘录、凭依华中为红色，在人妖名鉴中为黑色。 她左手手腕戴着镣铐，连着断了的锁链，整个右臂均为绷带缠成的空壳。 深秘录&凭依华 茨歌仙 人妖名鉴 人际关系 华扇经常到 博丽神社 指导与帮助 博丽灵梦 的工作，甚至使灵梦很烦她。 尽管并非负责华扇的 死神 ， 小野塚小町 却常去华扇的家 茨华仙邸 ，把茨华仙邸当成 用来休息的秘密基地 [ 茨歌仙46话 ] 。 与 伊吹萃香 、 星熊勇仪 同为 “山之四天王” [ 茨歌仙49话下 ] 。 华扇一直在避着萃香，防止她认出自己的真实身份。 她与勇仪仍然维持着不错的关系， 曾为了提升宴会上所用的酒的品质拜托勇仪把酒瓮的内部翻新 [ 茨歌仙16话 ] 。 自发监视 [ 凭依华 ] 着 宇佐见堇子 ，为此而与堇子交好。

霞网是为了收集空中漂浮的霞张开的。霞是仙人的主食，能卖不错的价格。发明这个装置的便是河童。 如果能捕到霞卖掉的话确实能赚很多钱。但是，因为网的缘故，也有很多鸟类被误捉了。协议会在这点上吵得很热闹。 住在妖怪之山上的仙人，茨木华扇小姐如此说道。 “啊，不，霞什么的……原来是用网捕捉的东西吗？先不管那个，如果野生的鸟会受到伤害的话，就应该停下来吧？” 河童始终坚持说那只要改良成不会让野生的鸟受到伤害的网就可以了，她们有捕捉霞的权利。 华扇小姐关于这点如此说道， “我也不是只吃霞的。而且食物什么的随地都有的……会有人特地买吗？” 协议会的结论是“禁止使用对飞天的生物有影响的网”。虽然没过多久河童就研究出了只会捕捉到霞的网，但由于买霞的人没有预想的多所以霞网照理说应该会变得越来越少吧。但是，能够有效率地捕捉到霞的这种技术肯定会在某些别的地方派的上用场的。 （射命丸文） 查看日文原文 东方求闻口授（访谈） —— 幻想乡缘起里也没有茨华仙。 ZUN
