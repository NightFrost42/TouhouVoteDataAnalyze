她本是被猴山驱逐的孤猴。 [ 兽王园 ] 后来，她听信了 他人 的妄言，认为自己是孙悟空转世，将曾掌控 圣域 的 幽灵 吞入体内成为猿神。之后，美天被 日白残无 引荐给 吉吊八千慧 ，就此成为了鬼杰组的一员。 她的设计参考了中国四大名著之一的《西游记》中家喻户晓的 孙悟空 。 她的立绘剪影与 天火人血枪 和 三头慧之子 一同出现在兽王园的封面上。 在兽王园之后，美天尚未在官方游戏中再次登场过，在官方出版物中则偶有出场。 生活状况 美天原本是被猴山驱逐的孤猴， 被 日白残无 怂恿，吞噬 圣域 的 幽灵 [ 兽王园 ] 后觉醒成为猿神。 她目前是 鬼杰组游击队员 [ 兽王园 ] ，但生活在幻想乡内 妖怪之山 上 圣域的森林 [ 兽王园 ] 里。 她在鬼杰组中拥有设定，如 “是从被雷劈中的石头里诞生的” [ 兽王园 ] 。 她认为自己是孙悟空转世，自称 “幻想乡的齐天大圣” [ 兽王园 ] 或 “圣域的齐天大圣”，但是这其实只是在开玩笑 [ 兽王园 ] 。 在摆脱残无的影响、变得清醒后，她想要成为圣域中前所未有的创新的妖怪，于是开始酿炼酒，还打算将炼酒向外卖。 [ 醉蝶华 ] 角色能力 美天拥有 操纵野生猿猴程度的能力 。 她现在已经成为 圣域 猴群的首领，将猴群中其他猿猴视为自己的伙伴，猿猴有事便会向她报告，她还会带领猿猴们酿炼酒。 [ 醉蝶华 ] 由于曾将 幽灵 吞入腹中，化为了 鬼 ，她觉醒了鬼的力量。 [ 兽王园设定文档 ] 外貌特征 美天有着一头及胸棕发，用金色头绳绑着下双马尾，头戴与孙悟空相似的紧箍，但 只是单纯的装饰，并没有用 [ 兽王园 ] 。 她有着红色的眼瞳。 她衣服的样式、风格、菱形镂空与住在 圣域 的 坂田合欢 类似，仅颜色变为上绿、中黄、下蓝，裙摆变为过膝且只有左臂穿着白色袖子。此外，右手手腕、左手袖子上沿均有金箍。 裸足。 她身后长着一条长长的棕色猴尾。 她裙摆和右手手腕金箍周围有黄色夹子形装饰， 样子很像薯条，甚至被 高丽野阿吽 认成了薯条 [ 兽王园 ] 。 她手持装有红色和紫色物体的细长试管，虽说如此，但它实际上 两端都有开口 [ 早苗线结局插图 ] ，美天时常用尾巴缠绕住它。 兽王园 人际关系 美天被 日白残无 怂恿，吞噬 圣域 的 幽灵 成为猿神。残无还将美天引荐给 吉吊八千慧 。 吉吊八千慧是美天的上司。但实际上美天听命于残无，是接受了残无的命令而成为八千慧的部下。不过最后她还是自愿为八千慧效力。 她自称讨厌 三头慧之子 、 高丽野阿吽 等犬类，但只是在迎合猴子和狗不共戴天的说法表演。 在还是普通猴子的时候她曾多次造访 守矢神社 ，单方面认识 东风谷早苗 。 登场剧情 官方游戏部分 在 东方兽王园 中，被 日白残无 怂恿吞噬 圣域 的 幽灵 成为猿神后加入鬼杰组，但仍作为残无的手下为残无效力，同时有意与 东风谷早苗 、 高丽野阿吽 混熟。 官方出版物部分 在 东方醉蝶华 中，带领猿猴们酿造炼酒和驱赶被吸引来的鸟，将一坛炼酒送给救助了猿猴的 奥野田美宵 来报答美宵，与来圣域调查“鸟地狱”的 博丽灵梦 战斗一番后将真相告诉了灵梦。 以下为按时间梳理的具体剧情。 剧透提示:
