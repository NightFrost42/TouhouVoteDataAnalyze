（神灵庙、求闻口授、人妖名鉴、幻存神签） 丰聪耳神子（官方角色，新作） - 新作角色。尸解复活的圣德太子，立场偏向人类的天才道士 角色介绍 丰聪耳神子是东方Project系列中的官方角色，首次登场于 东方神灵庙 ，担任神灵庙的六面Boss。 她是圣人，生在马厩里的她是个打小就被人承认天赋异禀的天才，一个远远超越了人类的超级人物。之后她通过尸解与复活，成为了尸解仙。 她在幻想乡所代表的宗教是道教，在幻想乡的立场偏向人类一方。 神子的设计原型参考了日本古代飞鸟时期尊奉佛法，留下了很多的逸话与传说的著名政治家 圣德太子 。她的立绘剪影出现在了神灵庙的封面上。 ZUN因为圣德太子最近突然开始被人们认为是个架空的人物而将他搬进幻想乡 。 [ 神灵庙Music Room ] 她性格洒脱轻浮，老爱开玩笑。但当直接与她面对面时不光容易露出纰漏，甚至有全部都被看透一样的感觉，会变成更多是敬畏的感觉。 [ 口授 ] 她本性正直，反感邪恶事物与妖魔邪道，内心想消灭世间的恶。会对幻想乡的人类及整个幻想乡的安危着想。 她自我意识非常强烈，自信而争强好胜， 在让自己看起来非同凡响这一点上跟 圣白莲 很相似，不过神子更有非凡感，登场后也在努力做非凡的事 [ 人妖名鉴 ] 。 鬼人正邪 认为神子 是个正论而独善，对自己宽容却不放过他人的不正当行为的家伙 [ GoU ] 。 在神灵庙之后，神子还作为自机登场于 东方心绮楼 、 东方深秘录 、 东方凭依华 中，在官方出版物中也时有出场。 生活状况 神子原本复活所在的 神灵庙 在 外面世界 被佛教僧侣封印，不久前神子把灵庙搬入幻想乡内 [ 神灵庙设定文档 ] 1 。在 神灵骚动 后，神子创造了一个仙界并将 神灵庙 迁至此处，建立的新的道场，之后大部分时候都在此修行至今。 人类村落 中想拜她为师修行仙术而前来的人十分之多，但似乎都只被神子用于打杂。 [ 茨歌仙18话 、 口授 ] 尽管早已成为了超越凡人的仙人，但却还没完全对俗世的事断绝来往，偶尔还会介入人类社会的事，如 贩卖四猿酱 与 向村民演讲 。 她也会去调查、解决一些异变。 角色能力 神子拥有 能够同时听十人讲话程度的能力 。 这项能力尽管听起来很是朴素的能力，始终是得天独厚之物，是神子生下来就是超人的地方。她做得到 同时听数个人讲话后不只都记得，更能理解并作出一针见血的回答 [ 神灵庙设定文档 ] 。 另外即使对方没直接表达出来，也能从对方说话的方式、表情、行为等读出对方所思所想。 当那个欲望成为形成对方本体的重要因素的时候。她的能力是接近于读心，差异的地方是在于，后者是读出对方的记忆以及当前的思考内容，而前者却能做到读出对方的资质以及他的未来。 这个能力带来的极强洞察力只不过是能体现神子超乎常人的能力之一，由于有着修习而来的道术傍身，她的实际力量不可限量。 作为 仙人 ，她拥有控制空间的仙术，她用这种仙术可以 瞬间移动到任何地方 [ 神灵庙 ] 和 破除永恒回廊的法术 [ 深秘录 ] 。 外貌特征 神子有着一头黄褐色及肩短发，头顶有形似猫头鹰耳朵的两簇头发，头戴 一边写有“和”字，一边写有“尊”字 [ 醉蝶华52话 ] 的紫色 耳罩 [ 口授 、 人妖名鉴 ] （一说 耳机 [ 神灵庙采访 ] ）。 她有着黄褐色的眼瞳。 神子出生在马厩里，打小就被人承认天赋异禀的天才，一个远远超越了人类的超级人物。 传闻她孩提时代就曾经用白胶木造出过四天王像，并且兴建了四天王寺。 [ 口授 ] 小时候，差人们经常把自己的各种琐事讲给她听，她能全部理解，并马上给出一针见血的指示。这被传出去后，她被人们尊为圣人，变得家喻户晓。 她长大后成为执政者，立下了显赫的功绩。听闻神子功绩而决定利用神子的 霍青娥 向神子进言信仰通过与自然融为一体来实现不老不死的道教，这使不满人类难逃死亡的命运的神子动了心。 于是作为执政者的神子便表面信仰与推广佛教以剥夺当权者以外之人的权力，背地里则一刻不停地进行着道教的不老不死研究。 之后她发挥着超于常人的能力，留下了各种各样的传说。但因为神子一直研究着不老不死的方法，用到的材料侵蚀了身体，神子觉得自己已经时日不多，便决意成为尸解仙，最后她与 物部布都 、 苏我屠自古 两位部下长眠于世间。 神子长眠后，有力量的佛教僧侣们一直封印着神子的灵庙，导致神子一直无法复活，神子的复活计划因而落空，她只好继续长眠。 很久之后，世间有人开始散播“神子的传说都是杜撰出来的”，而且现如今再也找不到有超乎常人能力的人了，大家觉得她那些伟业全部是虚构也无可厚非。于是神子便把灵庙搬到了 幻想乡 。那时的幻想乡连一座寺庙都没有，在这里，她可以随时准备复活。 但不久后， 命莲寺 建在了灵庙的正上方，压制住了神子的复活。 在 东方神灵庙 中，神子成功复活。同时，欲望具现化为神灵，神灵本能地感知到了神子的复活，而有欲望的神灵则自行聚集到神子的灵庙。 自机察觉神灵的异常，在调查中到达神子的灵庙，神子在聆听了自机的欲望后主动要求自机试着打倒自己，于是作为六面Boss被自机打败。 战后，神子 创造了一个自己的仙界 [ 口授 ] ，将 神灵庙 迁移至此处，并建立起了新的道场。 在
