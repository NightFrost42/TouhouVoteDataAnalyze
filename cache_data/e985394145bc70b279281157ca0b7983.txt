（求闻史纪、人妖名鉴） 梅蒂欣·梅兰可莉（官方角色，新作） - 新作角色。铃兰田里新诞生的妖怪，操纵毒的人偶 角色介绍 梅蒂欣·梅兰可莉是东方Project系列中的官方角色，首次登场于 东方花映塚 。 她本是被扔到铃兰田的人偶，天长日久而在几年前妖怪化，因为年轻而经验和知识严重不足。 她在官方游戏与出版物中登场次数不多，而且跟剧情没有什么联系。 生活状况 梅蒂欣生活在 无名之丘 的铃兰田中， 在花之异变时第一次离开铃兰田 [ 花映塚 ] 。 八意永琳 认为， 梅蒂欣用铃兰的毒素为自己创造了躯体而变成了妖怪 [ 花映塚 ] 。因此，她 可以自由自在地行动和思考全是靠铃兰的毒 [ 花映塚 ] 。 她很希望去解放人偶，但几乎没有人偶支持她。 [ 花映塚设定文档 ] 角色能力 梅蒂欣拥有 操纵毒素程度的能力 和 使用毒素程度的能力 。 稗田阿求 认为这是她 由于长期被放置在开着铃兰的无名之丘而掌握的能力 [ 史纪 ] 。 她可以使用以植物毒素为首的各种各样的毒，甚至包括碰到就会腐烂的毒、只是呆在下风位就会倒下的毒、留下后遗症的毒、长时间潜伏的迟效性毒，使得中毒后无法以确定的方法治疗。 从 由植物中取得的某种成分可以使动物狂乱 [ 花映塚设定文档 ] 的经验中，她学会了使用毒操纵人类。 一切毒都对她无效。 她自身也带有毒性，触摸她的身体会导致皮肤溃烂。 外貌特征 梅蒂欣的外表是用作表演腹语术的大型人偶的样子。原本的人偶也被认为是这般大小。 [ 史纪 ] 她有着一头黄色齐肩短发，头上带着红色缎带，扎成蝴蝶结。 她有着蓝色的眼瞳。 她上身穿着黑色短袖，领子为白色。在花映塚和人妖名鉴中身前和袖子上有类似缝线的红色花纹，在求闻史纪中则是类似于标签的扣子。戴着红色领结。在求闻史纪中右手腕上戴着一个有两个类似标签的装饰的手环。 她下身穿着红色长裙，中间有类似缝线的黑色花纹，裙摆打着褶子。背后系着白色大蝴蝶结。 她在求闻史纪中穿黑色鞋子，腿上和鞋上缠着缎带；在人妖名鉴中穿着白色短袜和红色圆头鞋。 她身旁常有一个小人偶，外表和梅蒂欣相似，但是头上的缎带改为蝴蝶结头饰，衣服的黑与红与梅蒂欣相反，背后多出两对类似于花映塚道中妖精的翅膀。在人妖名鉴中穿白色短袜，不穿鞋；在求闻史纪中还有单马尾；在人妖名鉴中穿黑色鞋子。
