人物名 伊吹萃香 日文名 伊吹　萃香（いぶき　すいか） 英文名 Ibuki Suika 种族 鬼 能力 操纵密和疏程度的能力 （萃梦想、幻存神签） 操纵密度程度的能力 （求闻史纪、人妖名鉴） 伊吹萃香（官方角色，新作） - 新作角色。嗜酒如命的鬼王，天真开朗而喜欢宴会 角色介绍 伊吹萃香是东方Project系列中的官方角色，首次登场于 东方萃梦想 。 她是在鬼族于幻想乡消失很长一段时间后，再次出现的 鬼 。 因为名字在日语中与西瓜谐音，也常常被爱好者们称作“ 西瓜 ”，有相当多的 二次创作 将伊吹萃香与西瓜联系在一起。 东方萃梦想 正式版的可执行程序里保留了体验版的西瓜图标，可以在更改文件夹或快捷方式图标时选择游戏可执行程序后见到。 幽幽子也用过这个neta。 [ 绯想天 ] 性格很开朗且喜欢酒。天真而且非常单纯。和一大群人在一起，会使气氛变得很快活，而只有两个人在一起，长时间的话会逐渐变烦。或者说非常孩子气。 [ 萃梦想设定文档 ] 不怎么说谎，但 在鬼之中是 有些自私不那么诚实的 [ 萃梦想 ] 。 她曾和 二岩猯藏 给 奥野田美宵 设套。 [ 醉蝶华7话 ] 非常 喜欢热闹 ，总是找别人开宴会。 [ 史纪 ] 虽然醉醺醺的，但 很 直率 [ 人妖名鉴 ] 。 在萃梦想之后，萃香还作为自机登场于 东方绯想天 、 东方非想天则 中，在官方出版物中也时有出场。 生活状况 萃香 曾经居住在地底，目前定居在幻想乡 [ 地灵殿设定文档 ] ，但居所不明，行踪不定。 其住处常被称作“鬼所居住之国” [ 萃梦想设定文档 ] 。 日常生活行迹未知，但遇到某些事态会进行 调查 [ 地灵殿设定文档 ] 。 虽然是鬼族的一员、山之四天王之一，但现在萃香并不会捉走人类，而是更喜欢 找人类、妖怪开宴会 [ 史纪 ] 。她认为 即使自己想袭击人类，也不可能明目张胆在 人类村落 袭击 [ 醉蝶华 ] 。 整日都在 喝酒 ，随身携带着名为 伊吹瓢 的酒葫芦。一般情况下都是醉醺醺的。 她是 博丽神社 宴会上的常客，在 蚕食鲵吞亭 开业后也常去蚕食鲵吞亭喝酒。 角色能力 萃香拥有 操纵密和疏程度的能力 和 操纵密度程度的能力 。 通过运用这种能力，她既可以聚集起细小的灰尘或庞大的物体，也可以通过控制密度， 让密度变得稀薄使事物变为雾态，或加大密度使事物生热，进而熔解 [ 史纪 ] 。 她还可以 对自己施加这个能力 使自己雾化 [ 文花帖 、 萃梦想 ] 。通过雾化，她可以 避开攻击 [厌雾] 或者监视他人的行动，当然也可以 进行攻击 [ 茨歌仙9话 ] 。 运用该能力，她还能 从呼吸中制造大量小型 分身 [ DS ] ，以及 让自己的身体 巨大化 [ 符卡 ] 。 这项能力也可以对比较抽象的“密度”进行操纵，比如 将众人聚集起来开宴会 [ 萃梦想 ] 。 她的肉搏能力强大。 作为 鬼 ，萃香拥有与体形不符的、远超人类的身体素质，她的 力量 之大令人咋舌。 她 曾经 打碎过倒映月亮的天穹 [ 东方文花帖（书籍） ] ，能 举起并投掷重物——甚至远比自己体形大的巨石 [ 符卡 ] ，还能 将对手抓住并不断跳跃、摔打 [ 符卡 ] ，甚至放出过 能凭一己之力毁掉妖怪之山的豪言壮语 [ 史纪 ] ，对自己的力量很有自信。 她身上的 锁链 也可以被投掷出与束缚住对手，并且 能够驱散灵力，甚至使对手持续损失灵力 [ 符卡 ] 。 在 东方非想天则 中，她普通攻击连段的伤害第二高。 作为鬼，她 酒量很大 ，无时无刻不在喝酒，但说话却又显得清醒直率。 她能 从口中喷出灼热的酒气 [ DS ] 。 她拥有的酒葫芦“ 伊吹瓢 ”是鬼宝，看上去很普通，却能无限涌出酒。 [ 萃梦想 ] 伊吹瓢有预防翻倒的机能，使得酒不能一下子全部涌出来。 [ 萃梦想 ] 伊吹瓢中的酒水似乎能给鬼之外的种族造成幻觉甚至影响记忆 。 [ 醉蝶华 ] 萃香称伊吹瓢能无限制涌出酒来是因为她 往伊吹瓢里浸入过酒虫的精华 [ 三月精S ] 。 但后来她又称伊吹瓢的酒是 幻想乡里所有酒葫芦中自然挥发出来的酒聚集而成的 [ 醉蝶华 ] 。 伊吹瓢里的酒 很难喝，并且能像鬼杀酒一样让鬼都喝醉 [ 醉蝶华 、 二轩目 ] 。 不过， 二岩猯藏 曾称伊吹瓢内倒出的酒为“好酒”“美酒”。 [ 茨歌仙 ] 伊吹瓢也可以用来进行攻击。 外貌特征 萃香肤色偏白，身高体型接近一般幼童，形象不同于传统的 鬼 。 ，左侧的角上系着紫罗兰色的布片饰品，右侧的角上除兽王园外有白色螺旋状线形物体缠绕，在兽王园中则缠绕着绿色的藤蔓。 她在兽王园中有着红色的眼瞳，在除兽王园外的其他作品中有着橘棕色的眼瞳，在史纪、人妖名鉴中左侧长着虎牙。 她上身穿着白色无袖衬衫，袖口像被撕开了一样参差不齐，脖子上用红色布带系着蝴蝶结。 左手锁链连接着黄色圆球，右手锁链连接着红色三角锥，分别象征着无与调和 [ 东方书谱 ] 。 兽王园 萃梦想 绯想天&非想天则 求闻史纪 ZUN设定画 人妖名鉴 人际关系 萃香和 茨木华扇 [ 茨歌仙 ] 、 星熊勇仪 [ 地灵殿 ] 与 八云紫 [ 萃梦想 ] 是旧识，和勇仪、华扇同为 山之四天王 之一。曾经四天王一同住在 妖怪之山 ，现在已经各自分散。 因为常常去 博丽神社 的原因，萃香和
