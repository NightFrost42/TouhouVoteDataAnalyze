有着爱讲理的性格，无论碰上什么事总要抱怨几句。她还是光之三妖精中最冷酷的一个，并会在什么地方迷上人类等不自然的东西 [ 上海爱丽丝通信 ] 。 除三月精外，露娜还作为Boss登场于 妖精大战争 中，在其他官方出版物中也时有出场。 生活状况 光之三妖精 一起住在 魔法森林 之中的“自亘古以来便矗立着的大树” [ 上海爱丽丝通信 ] 内 三妖精的家 中，后光之三妖精搬家到 位于博丽神社内外分界线 [ 三月精S ] 的 大水楢树 [ 三月精O ] 中。 由于性格原因， 露娜的房间从墙壁到床上都覆盖着以黑色和白色为基调的蕾丝类的织物，其间还以丝带作为装饰。房间中的其他地方也都塞满了捡来的人偶和毛绒玩具之类的东西，有一种不自然感 [ 上海爱丽丝通信 ] 。 光之三妖精会一起进行恶作剧， 但三人没什么配合 [ 上海爱丽丝通信 ] 。露娜 因为能力在其中担任主力，因此通常由她殿后，由于她很迟钝并过于依赖消音能力也通常是她被落单和抓住 [ 上海爱丽丝通信 、 史纪 ] 。 她常在夜晚单独活动。 [ 史纪 ] 喜欢在十六日之夜 1 外出，收集从月亮上掉落的东西，想把自己的房间变成“地上的月亮”。 [ 三月精E ] 她喜欢阅读报纸，在看报纸时常常戴上眼镜。 角色能力 露娜拥有 消除周围声音程度的能力 。
