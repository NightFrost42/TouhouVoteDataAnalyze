英文名 Komeiji Koishi 种族 觉 能力 操纵无意识程度的能力 （地灵殿、求闻口授、人妖名鉴、幻存神签） 古明地恋（官方角色，新作） - 新作角色。放弃读心而闭锁内心的觉，处于无意识状态 角色介绍 古明地恋是东方Project系列中的角色，首次登场于 东方地灵殿 ，担任地灵殿的EX面Boss。 她是觉妖怪，一种拥有读心能力的妖怪，但恋封闭内心后失去了该能力，同时获得操纵 无意识 的能力。 [ 地灵殿设定文档 ] 恋的角色设定与心理学有着非常深厚的渊源，例如潜意识/无意识、本我与超我、梦境等等概念，许多符卡名都与著名的心理学研究者以及其学说有关，这在东方Project官方角色中是非常独特的。 从 东方心绮楼 开始，ZUN有意推一批角色，但只有恋的人气被推上去了。 [ ZUN×海原海豚 ] 恋是矛盾的角色，是作为没有角色性的存在而诞生的，是对角色性的否定。 [ 人妖名鉴 ] 她虽然试图通过回避他人的恶意来避免成为被厌恶者，但她 依然含有很多理应遭到厌恶的东西，心怀某种黑暗 [ 人妖名鉴 ] 。这种矛盾也体现于性格上：恋 性格空洞，难以交流 [ 口授 ] ，说话的风格忽而可爱忽而黑暗。 在地灵殿之后，恋还作为自机登场于 东方心绮楼 、 东方深秘录 、 东方凭依华 中，在官方出版物中偶尔出场。 生活状况 恋居住在 旧地狱 的 地灵殿 ，也会 时不时造访地上 [ 口授 ] 。 恋一直处于无意识的状态，以至于她本人也不知道接下来她自己会做什么， 无心无欲地顺从着周遭的环境生活着，就像随风飘舞的布片一般，没有任何目的，仅仅只是在周围四处游荡 [ 地灵殿设定文档 ] ， 更多地是凭借自己当时的感觉行动 [ 口授 ] 。对于她自己而言这种生活也算是有趣，但周遭看来这样的她稍微有一点可怜。 恋虽然使用有关梦境的符卡（ 表象「列祖列宗入梦来」 、 「胎儿之梦」 、 梦符「先祖大人在看着你」 ），但是她 却是不做梦的，心灵非常空洞 [ 凭依华 ] 。 恋的姐姐 古明地觉 给她分配了几只宠物，恋有时会和宠物们一同玩耍 [ 地灵殿设定文档 ] 。不过她和她的姐姐不同，并 不讨动物喜欢，但也不被讨厌 [ 口授 ] 。 在遇见地灵殿的自机后，原本心境空空如也的恋 逐渐有了一些变化 [ 地灵殿设定文档 ] 。 她开始 在意大家有没有无视她 [ 心绮楼 ] ， 尝试着和地上的其他人进行接触 [ 深秘录 ] ， 能够和其他人组队调查异变 [ 凭依华 ] 等等。 她在 命莲寺 的劝诱下入了佛门，但没有出家。 [ 口授 ] 角色能力 古明地恋拥有 操纵无意识程度的能力 。 对于她本身来说，这使得她可以在 无意识 的状态下行动， 简单来说就是“自己不知道自己接下来会做什么”、“顺从现场气氛行事”这样的状态。没有内心，没有欲望。 [ 口授 ] 不过她似乎也会 希望受到别人的关注 [ 心绮楼 ] ，这可能与她心境的变化有关。 恋所处的这种无心无欲的状态，被 云居一轮 认为是无我的境界。 [ 心绮楼 ] 圣白莲 认为她舍弃了内心从而临近了“空”的境地。这对于积累了修行的僧侣而言也是很困难的事情。她或许不需要修行也能接近觉悟的境界。 [ 口授 ] 但白莲后来认为有无心和什么都不想是有区别的。 [ 凭依华 ] 在无意识下行动的恋几乎不会被察觉 [ 地灵殿设定文档 ] ，甚至可以做到 不受阻拦地穿过守卫森严的天狗领地 [ 地灵殿 ] 。 这使得 看不到她就不能察觉到她的存在，看到她也同样感觉不到她的存在，她一旦离开视野就会立刻忘掉她 [ 口授 ] ，就像她的名字谐音“小石头” 1 那样。 这样的她 虽然不再招人讨厌，但同时也不会被恐惧，也不会被人喜欢了 [ 地灵殿设定文档 ] ，但 也有一部分她的仰慕者存在 [ 口授 ] 。 特殊条件下，她会更容易被感知到，但那 依赖于外来的力量 [ 心绮楼 ] 。 在 宗教战争 期间，恋捡到了具有希望之力的希望之面。加上她积累的人气，她变得极容易被感知， 以至于会被别人主动要求对战 [ 心绮楼 ] ；在 完全凭依异变 期间， 恋处于 可以被清楚地看到 [ 凭依华 ] 的状态。 大人无法看见恋，只有一部分小孩子能看见，恋就如在孩提时代曾经一起嬉闹玩耍过，但当长大后就完全遗忘的存在，就是所谓的“幻想中的朋友”。 [ 口授 ] 古明地觉 作为能读心的觉，可以认知恋的存在，但也无法窥视恋的内心。 [ 口授 ] 不过这种影响并非对所有人都有效，例如 铃仙·优昙华院·因幡 靠能力可以感知恋无意识的举动 [ 凭依华 ] 。 恋会 在无意识中准备好攻击并发出弹幕 [ 深秘录角色介绍 ] ，她 好像甚至也不会意识到自己正在发射弹幕 [ GoM ] 。 此外，该能力也可以作用于其他人。恋可以强行唤醒其他人在潜意识中思考的事情，或是唤醒其他人内心深处连记忆都称不上的某种印象等等。 [ GoM ] 恋因为害怕遭人厌恶闭上了自己的第三只眼，失去了读心的能力，这也代表着封闭了自己的内心，不接受别人的心灵完全将其屏蔽。 [ 地灵殿设定文档 ] 读心的能力代表了心灵的强度。 [ 地灵殿设定文档 ] 正因她这种逃避的处世哲学，她甚至会遭到弹幕的厌恶，使得子弹变得非常刺人。 [ GoM ] 在 都市传说异变 时，恋选择了「梅莉小姐的电话」这一都市传说作为自己的灵异。她会给附近的人打电话，并转移到其背后进行攻击。于是， 在恋的 二次创作 中以电话、菜刀为元素的作品也十分丰富 。 在都市传说异变已经结束很久之后，恋仍然将菜刀作为武器。 [ 智灵奇传 ] 外貌特征 恋有着一头浅绿色及肩短发，略卷，头戴黑色圆顶帽，上面系着一条黄色缎带，打着蝴蝶结。 她的瞳色在地灵殿、人妖名鉴中为深绿色，在心绮楼、深秘录、凭依华中为和头发颜色相同的绿色。 她上身穿着黄色长袖上衣，衣服扣子为蓝绿色，衣领为绿色，衣领外缘以及衣袖饰有墨绿的百褶。 她下身穿着印有 蔷薇 [ 地灵殿采访 ] 纹样的渐变绿及膝裙，腰部为白色，往下绿色加深，直至膝盖处为深绿色。 除口授外，她脚穿黑色短靴，在口授中她则脚穿黑色皮鞋和白色短袜。她鞋子上有紫色的心形饰片并链接着第三只眼的一部分。 她周身缠绕着 第三只眼 ，在地灵殿中表皮为深蓝色，在其他作品中为紫色，眼睑闭合，两侧延伸出两根类似血管的紫管子（在地灵殿中为由深蓝渐变为紫色），在空中绕出爱心状的图案。 地灵殿 心绮楼 深秘录&凭依华 求闻口授 人妖名鉴 人际关系 由于自己的能力，她很难和他人正常交流，也 没去建立复杂的人际关系 [ 口授 ] 。 她和姐姐 古明地觉 以及姐姐的宠物 火焰猫燐 、 灵乌路空 同居于 地灵殿 ，关系较为亲密。 觉非常牵挂封闭了内心的恋，她命令宠物们和恋一起玩耍，并给恋分配了几只专属宠物，希望恋能够通过饲养宠物找到目的，逐渐变得再次接受他人的心灵。 [ 地灵殿设定文档 ] 由于恋开始在地上活动，觉偶尔还会因为寻找她而前往地上。 [ 凭依华 ] 恋在与其他人交流的时候也会提及阿空与阿燐。 由于地灵殿地处 旧地狱 ，因此恋还与一些现在以及曾经在旧地狱居住的人相互认识。 伊吹萃香 认识她，并称她为“小恋（こいしちゃん）”。 [ 地灵殿 ] 恋认识曾被埋入地下的 云居一轮 、 村纱水蜜 等人。 [ 心绮楼 ] 而在 间歇泉骚动 发生后，恋逐渐接触并结识了一些地上人。她在 完全凭依异变 期间和 雾雨魔理沙 组队一起探索完全凭依的性质。 [ 凭依华 ] 魔理沙在和恋组队后觉得恋 直率又好使 [ 交叉评论 ] 。 在 宗教战争 中恋 因 地裂 掉到地底的 秦心 的希望之面 [ 心绮楼 ] ，但恋并没有打算还给心， 两人因此曾大打出手 ，最终心也没能从恋手里夺回希望之面。 后来，心得到了新的希望面具并开始锻炼自己控制感情，所以 不再需要从恋那里拿回原先的面具了 [ 心绮楼 ] 。 秦心称恋为 “宿敌” [ 心绮楼 ] ，两人会对 面具归属 [ 心绮楼 ] 等问题 以孩童般的口吻进行争论 [ 深秘录 ] ，也会 进行友善交谈 [ 凭依华 ] 。 登场剧情 官方游戏部分 在 东方地灵殿 中，为给自己的宠物求得神之力而前往 守矢神社 ，虽未找到给 灵乌路空 力量的二神却遇到自机，对自机感兴趣而与自机打了一架。 在 东方文花帖DS 中，被 射命丸文 和 姬海棠果 进行新闻取材。 在 东方心绮楼
