- 新作角色。冥界的亡灵，难以捉摸的任性大小姐 角色介绍 西行寺幽幽子是东方Project系列中的官方角色，首次登场于 东方妖妖梦 ，担任妖妖梦的六面Boss。 她是 亡灵 公主，西行寺家的大小姐，负责管理 冥界 的 幽灵 。 她性格 开朗，而且给人呆呆的感觉 [ 永夜抄Manual ] ， 十分慢性子 [ 史纪 ] ， 相当无法捉摸 [ 萃梦想角色介绍 ] ， 任性妄为。她行事轻浮 [ 永夜抄角色设定 ] ， 整天游手好闲，看不出所想的事情，总是做出让人很困扰的行动，却是一个真心爱着幻想乡之外的这个国家的爱国者 [ 萃梦想设定文档 ] 。不过， 尽管她看上去糊涂，但是她的言行举止从容冷静 [ 绯想天角色介绍 ] 。 在妖妖梦之后，幽幽子还作为自机登场于 东方永夜抄 、 东方萃梦想 、 东方绯想天 、 东方非想天则 中，作为一面Boss登场于 东方神灵庙 中，在官方出版物中也时有出场。 生活状况 幽幽子已经以 亡灵 的姿态存世千年以上，也将永远居住在 冥界 的 白玉楼 中。 [ 史纪 ] 她受阎魔之命，管理着冥界的幽灵。 [ 史纪 ] 她对人没有怨恨。 角色能力 幽幽子拥有操纵、统制死之后的 幽灵 的能力。 这项能力是 阎魔 命令她管理幽灵的原因 [ 史纪 ] 。 幽幽子原本只是能操纵死灵的人类。 [ 妖妖梦设定文档 ] 幽幽子自称 不管在什么地方都能把人魂灯点亮 。 人魂灯是 冥界 的道具，它发出的光，不管相隔多远，不管有没有障碍物，幽灵都可以看见并会被吸引过去。 [ 香霖堂 ] 幽幽子还拥有 操纵死亡程度的能力 。 幽幽子不知何时变得能够引诱死亡，并能很简单地将人置于死地。 她为摆脱这种能力而自尽。 [ 妖妖梦设定文档 ] 操纵死亡的能力无法抵抗。 [ 史纪 ] 然而， 蓬莱之药是幽幽子的“天敌”，这项能力对蓬莱之药和服下蓬莱之药的人力不能及 [ 永夜抄 ] ，所以对服下蓬莱之药的人 无效 [ 永夜抄 ] 。 死在幽幽子手上的灵无法成佛，只能待在冥界。 [ 永夜抄 ] 外貌特征 幽幽子属于 身长较高 [ ZUN的邮件答复 ] 的角色， 样子和是人类时没太大差别，只是肌肤和发色变得有些淡薄 [ 史纪 ] 。
