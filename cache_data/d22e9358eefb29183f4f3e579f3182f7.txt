，担任妖妖梦的三面Boss，并且由原有的名字“爱丽丝”变更为“爱丽丝·玛格特洛依德”。本条目仅说明 新作中的爱丽丝 ，在旧作中登场的爱丽丝请参考 爱丽丝（旧作） 。 她是 由人类修行而成 [ 求闻史纪 ] 的魔法使，擅长操控人偶的魔法，被称为 七色的人偶师 [ 东方妖妖梦 ] 。 爱丽丝 对于东方Project系列来说算的上是有点比较特别的角色 [ 东方妖妖梦 ] 。 东方妖妖梦的Music Room界面与结算界面的背景图均为爱丽丝。 爱丽丝 对他人一点都不关心，却对魔法很执着，尽管外表看起来坚强却有着弱小的一面 [ 萃梦想设定文档 ] 。她基本上不拘束于事物，性格上不固执不厌恶。她喜欢一个人独处，不过却常常参加宴会和在祭典上表演人偶剧。 她看上去像个大人， 非常好战，和她挑起胜负的话会立刻欣然接受 [ 史纪 ] ，但是爱丽丝 不喜欢以压倒性的力量战胜对方，所以总是用比对手稍微高一点的力量战斗，甚至哪怕打输都不以全力进行战斗，因为全力战斗输了后就没有退路了。这一点和 博丽灵梦 很像 [ 妖妖梦设定文档 ] ，也就是很重视保留实力。 在妖妖梦之后，爱丽丝还作为自机登场于在 东方永夜抄 、 东方萃梦想 、 东方绯想天 、 东方非想天则 中，在官方出版物中也偶有出场。 生活状况 爱丽丝平时独居在位于 魔法森林 的家 玛格特洛依德邸 中，喜欢一个人在家制作各种人偶， 正在尝试制造一种会自动进行保养的人偶 [ 永夜抄设定文档 ] ， 也会撰写魔法书 [ GoM ] 。因此，她不怎么爱出门，但是也会 去祭典上表演人偶技艺 [ 史纪 ] 。 她是个收集家，有着收集书籍之类的魔法道具的癖好，最近热衷于收集有说道的人偶。 [ 妖妖梦角色设定 ] 她 拥有来自各地的人偶 [ 妖妖梦里音乐评论 ] ， 家里摆满了大量的人偶 [ 萃梦想设定文档 ] 。 她平时的料理、洗衣、打扫等家事大都是她操纵人偶完成的。 [ 史纪 ] 尽管身为魔法使，她仍然保持着和人类一样的吃饭与睡觉的习惯。 [ 史纪 ] 角色能力 爱丽丝拥有 使用魔法（人偶操纵）程度的能力 。 作为人偶师，她的 灵巧度在幻想乡中首屈一指,可以宛如活着一样地操纵人偶 [ 萃梦想设定文档 ] 。 操纵人偶的时候，她须要全力以赴，因此行动缓慢并且战斗能力也低。 [ 史纪 ] 她所使用的人偶 大多是自己亲手制作的 [ 史纪 ] 。 她自称自己的 人偶都是自己做的而不是买来的 [ 三月精S ] 。 人偶用靠魔法活动的魔法线连接，以意志来远距离操控。除了制造人偶之外，其他人类做得到的活动爱丽丝的人偶都能完成，甚至包括操控人偶。 [ 史纪 ] 她有着 会自爆的人偶 [ 史纪 ] ， 甚至连给魔理沙支援对战与对话用的人偶里面都 装着火药 [ 地灵殿 ] 。 在格斗作中，爱丽丝在必杀技和符卡中大量运用会自爆的人偶。 她并没有什么格外擅长或不擅长的属性。 [ 妖妖梦设定文档 ] 她的体术、移动身体和使用魔法都很普通。 [ 萃梦想设定文档 ] 外貌特征 爱丽丝属于 身长较高 [ ZUN的邮件答复 ] 的角色， 有着白色的皮肤和人偶般的外表 [ 史纪 ] 。 她有着金色及肩短发，头戴红色发箍，在永夜抄和人妖名鉴中发箍有白色褶子。 她的眼瞳在妖妖梦、人妖名鉴中为蓝色，在其他作品中为褐色。 她身穿蓝色短袖连衣裙，在妖妖梦、萃梦想、永夜抄、绯想天、非想天则中及踝，在史纪、人妖名鉴中及膝。裙子门襟用一排一字扣系住，扣子在妖妖梦、永夜抄中为黑色，在萃梦想、绯想天、非想天则中为红色，在人妖名鉴中为白色。领子上系着红色领巾，领巾有白色褶边。红色肩披白色披肩，边缘在人妖名鉴中有两圈黑色细线条，在其他作品中有一圈黑色粗线条。 她腰间系着红色腰带，腰带有白色褶子，结系在身前。 在妖妖梦、永夜抄中，她脚穿白色短袜。在史纪、人妖名鉴中，她脚穿棕色中靴。 手持一本锁住的黑色封皮的魔法书，在妖妖梦、萃梦想、绯想天、非想天则、人妖名鉴为右手，在永夜抄、史纪为左手。 在永夜抄中身旁有一只穿着红色连衣裙、脑后戴着红色大蝴蝶结的金发人偶，在除妖妖梦外其他作品中身旁有一只穿着黑色连衣裙和白色围裙、脑后戴着红色大蝴蝶结的金发蓝瞳人偶。
