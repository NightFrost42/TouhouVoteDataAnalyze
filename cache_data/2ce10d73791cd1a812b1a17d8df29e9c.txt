铃仙·优昙华院·因幡（官方角色，新作） - 新作角色。狂气的月兔，堕至地上并被永远亭庇护的逃兵 角色介绍 铃仙·优昙华院·因幡是东方Project系列中的官方角色，首次登场于 东方永夜抄 ，担任永夜抄的五面Boss。 铃仙曾是 月之都 中绵月家的月兔，在 四十多年前 [ 儚月抄 ] 人类开始向月球发动侵略战争时从月球逃走。在 大约三十年前 [ 儚月抄 ] 逃到幻想乡，被 蓬莱山辉夜 收养。 她的性格 疯狂与悠闲并存，绝算不上温柔。她偶尔会表现出悠闲的一面，但战斗时和日常时的性格截然不同。她自信较少，很少配合人类，完全是自顾自的改变自己的性格 [ 绯想天设定文档 ] 。她的 性格比较胆小随便，合作能力差 [ 儚月抄 ] 。 在永夜抄之后，铃仙作为自机在官方游戏中多次出场，在官方出版物中也时有出场。 生活状况 铃仙作为 蓬莱山辉夜 的宠物居住在 永远亭 里，过着安稳的生活。 她能对 因幡天为 下命令，被阿求认为 是真正的兔子们的领袖 [ 史纪 ] 。 但是 地上的兔子们只听天为的命令 [ 儚月抄 ] ，铃仙不能直接指挥兔子，天为也基本不听铃仙的话。 她 有时会伪装成药贩到人类村落卖药 [ 铃奈庵 ] 和维护村民们的药箱。卖完药后就会 像避开人类似的消失在迷途竹林里 [ 史纪 ] 。 角色能力 铃仙作为月兔，拥有 操作波长程度的能力 和 无视距离感知其他月兔发出的兔之波动（念波）的能力 [ 永夜抄 ] ，而且这两项能力都十分优秀，因而 绵月依姬 评价她 有才能得多 [ 儚月抄 ] 。 铃仙操纵波长的能力，体现为她可以 操纵寄宿在事物之上的波 [ 口授 ] 。 事实上，这项能力可以操控包括光波、声波、 空间的波、构成物质和生物的波、构成生物的思考和感情的波 [ 绯想天设定文档 ] 在内的波的波长、振幅、相位，达到使人变轻松或疯狂、让存在过剩或稀薄、使事物无法被触碰甚至外形完全消失等多样的效果。 改变波长的影响范围可以很大。 她曾通过改动 迷途竹林 的波长，使整片竹林变得无法离开。 [ 三月精S ] 运用这项能力，铃仙 能够轻易地隐藏身形和扰乱人心 [ 儚月抄 ] 。 铃仙还可以发射 视线波 [必杀技Ripple Vision] 、 思念波 [ 符卡 ] 、 特殊波长 [ 符卡 ] 来扰乱认知或进行攻击。 此外，使用这项能力，她可以轻松地 和老鼠对话 [ 铃奈庵 ] 。 铃仙的 操纵疯狂程度的能力 需要靠前述的操纵事物波长的能力，还与寄宿在红色瞳孔中的巨量疯狂有关。 这种能力与 月之狂气和使来月球的人疯狂的催眠术 [ 永夜抄 ] 有关。 注视铃仙的红眼与注视满月有相同的效果 [ 史纪 ] ，即使人发狂。 运用这个能力，铃仙可以使敌人 辨别不出方向，只会摇摇摆摆地飞行 [ 永夜抄 ] 。 铃仙所引起的幻觉直接作用于脑，甚至比光还快。 [ 非想天则 ] 铃仙可以从眼中发出狂气的波长或视线波，以 干扰认知 [ 符卡 ] 或进行 直接攻击 [必杀技Under Sense Break] 在 凭依华 中，狂气可以用来加成攻击效果。 感知其他月兔发出的兔之波动，可以用以与远方的玉兔通信。 但是，一般的月兔只能做到简单的意思沟通， 无法正确地交谈，只能说是大家平常所想的事如风闻般传进耳中 [ 儚月抄 ] 。而铃仙却能接收针对自己的内容丰富的信息，例如 永夜异变 前告知情况和要求铃仙返回的信息 [ 永夜抄 ] 。 铃仙还可以 从食指发射子弹，而且子弹可以带有精神效果 [ 符卡 、必杀技Mind Explosion] 。 铃仙自称会分身术。 [ 深秘录 ] 相比于 妖兽 ，铃仙移动较缓慢。 [ 史纪 ] 外貌特征 ZUN设计铃仙时有意 表现出千篇一律毫无个性的感觉 [ 人妖名鉴 ] 。 她 有着别的兽类很少有的直立姿势 [ 史纪 ] ， 和 雾雨魔理沙 差不多高，比 博丽灵梦 矮一点 [ 秋例11对话 ] 。 她有着一头浅紫色及踝长发，头上有一对皱巴巴的、从中间开始弯折的长长的兔耳朵，看起来像是人造品， 但其实是真的兔耳 [ 秋例11对话 ] ，且她 同时长着人耳和兔耳 [ 绀珠传结局图 ] 。在永夜抄、花映塚、凭依华中，兔耳朵根部长有像纽扣一样的东西。 耳朵能显现出压力，压力大就会变皱。 [ 绀珠传结局图 ] 作为白兔，她有着红色的眼瞳。 她上身穿着白色衬衫，在花映塚中和穿外套时为长袖，其他作品中为短袖。打着红色领带，除永夜抄外的作品上面别着一个胡萝卜形状的领带夹，人妖名鉴中还别着一个兔子领带夹。在永夜抄、求闻史纪、儚月抄和人妖名鉴中她在衬衫外穿着黑色的制服外套，左侧有一个月牙形的胸针， 现在已不再使用 [ 秋例11对话 ] 。在永夜抄中，外套口袋上露出白色手帕。 她下身穿着短裙，在不同作品中有白色、红色、蓝色之别。 她脚穿棕色皮鞋，在史纪中穿着白色过膝袜，在其他作品中脚穿白色短袜。 她身后长着兔尾巴，在非想天则中为淡紫色，在深秘录、凭依华中为暗蓝色。 在深秘录、绀珠传、凭依华中，她手持喇叭状枪械，枪口有兔耳装饰。 铃仙的2P色，将红色改为黄色，白色改为黑色，领带夹也改为黑色。 永夜抄 花映塚 花映塚2P 绀珠传 绯想天&非想天则 深秘录&凭依华 求闻史纪 儚月抄 人妖名鉴 人际关系 铃仙曾是 绵月依姬 和 对于此番言论神社方面，「吃兔子的肉到底哪里野蛮了啊。那东西从古至今不是一直都在吃吗。还是说兔子本身就是野蛮的？不过那种毫无意义的在野外乱跑的行为也许真的挺野蛮的呢」（灵梦） 如上，保持不愿意停止使用兔肉的态度。 兔角同盟是由地上的妖怪兔们结成的，以身为月兔的铃仙·优昙华院·因幡为领袖，为创造兔子的理想乡而活动的集团。负责领导非妖怪的地上兔子的是因幡天为干部（兔子）。她针对此事发表如下看法。
